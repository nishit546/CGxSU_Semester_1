
## Level 4 – Strings

Assume inputs are strings like `"hello"`.

---

### Q31. Length of String

Print length of a string.

* Input: `"hello"` → Output: `5`
* Input: `""` → Output: `0`
* Input: `"JavaScript"` → Output: `10`

---

### Q32. Print Each Character on New Line

Given a string, print each character on a new line.

* Input: `"cat"` → Output:
  `c`
  `a`
  `t`

* Input: `"hi"` → Output:
  `h`
  `i`

---

### Q33. Count Vowels

Count vowels (`a, e, i, o, u` – you can decide if you treat uppercase as vowel too).

* Input: `"hello"` → Output: `2` (e, o)
* Input: `"JAVASCRIPT"` → Output: `3` (A, A, I – if case-insensitive)
* Input: `"sky"` → Output: `0`

---

### Q34. Count Consonants

Count consonants (letters that are not vowels).

* Input: `"hello"` → Output: `3` (h, l, l)
* Input: `"abc"` → Output: `2` (b, c)
* Input: `"a e i"` → Output: `0` (if you ignore spaces, only vowels)

---

### Q35. To Uppercase

Convert a string to uppercase.

* Input: `"hello"` → Output: `"HELLO"`
* Input: `"Js"` → Output: `"JS"`
* Input: `"123abc"` → Output: `"123ABC"`

---

### Q36. To Lowercase

Convert a string to lowercase.

* Input: `"HELLO"` → Output: `"hello"`
* Input: `"Js"` → Output: `"js"`
* Input: `"123ABC"` → Output: `"123abc"`

---

### Q37. Reverse a String

Reverse all characters in the string.

* Input: `"hello"` → Output: `"olleh"`
* Input: `"abc"` → Output: `"cba"`
* Input: `"a"` → Output: `"a"`

---

### Q38. Palindrome String

Check if a string is a palindrome.

* Input: `"madam"` → Output: `true`
* Input: `"racecar"` → Output: `true`
* Input: `"hello"` → Output: `false`

---

### Q39. Count Occurrences of 'a'

Count how many times `'a'` (or `'A'`) appears.

* Input: `"banana"` → Output: `3`
* Input: `"Apple"` → Output: `1` (if case-insensitive, a only)
* Input: `"xyz"` → Output: `0`

---

### Q40. Remove All Spaces

Remove all spaces from a string.

* Input: `"hello world"` → Output: `"helloworld"`
* Input: `"  a b  c  "` → Output: `"abc"`
* Input: `"no_spaces"` → Output: `"no_spaces"`
