## Objective

The goal of this assignment is to understand and implement:

* Client-side routing using **react-router-dom**
* Navigation using a common **Navbar**
* Fetching data from multiple public APIs
* Rendering API data on different routes
* Passing navigation via cards and links

---

## Project Overview

You will build a **React application** with **5 routes**:

1. Home
2. Food API Page
3. Fake Store API Page
4. Movie API Page
5. Recipe API Page

All routes must be accessible from the **Navbar** and through **Home page cards**.

---

## Routing Requirements

* Use **react-router-dom**
* Total routes required: **5**
* Each route must:

  * Have a unique URL path
  * Render a separate component
  * Fetch data from an API
  * Display data on the UI

---

## Route Details

### 1. Home Route (`/`)

**Purpose:**
Acts as the landing page of the application.

**Requirements:**

* Display a short introduction about the project
* Show **4 cards**, one for each API:

  * Food
  * Products
  * Movies
  * Recipes
* Each card must:

  * Contain a title and short description
  * Navigate to its respective route when clicked

---

### 2. Food API Route (`/food`)

**API Suggestion:**

* Any public food-related API

**Requirements:**

* Fetch food data using `fetch`
* Display:

  * Food name
  * Image (if available)
* Show data in a **card/grid layout**

---

### 3. Fake Store API Route (`/products`)

**API:**

```
https://fakestoreapi.com/products
```

**Requirements:**

* Fetch product data
* Display:

  * Product image
  * Title
  * Price
* Use reusable card components 

---

### 4. Movie API Route (`/movies`)

**API:**

* TMDB API (or any open movie API)

**Requirements:**

* Fetch movie data
* Display:

  * Movie poster
  * Movie name
  * Rating (if available)

---

### 5. Recipe API Route (`/recipes`)

**API Suggestion:**

* Any public recipe API

**Requirements:**

* Fetch recipe data
* Display:

  * Recipe name
  * Image
* Show data in a clean UI format

---

## Navbar Requirements

* Navbar must be visible on **all pages**
* Navbar must contain links to:

  * Home
  * Food
  * Products
  * Movies
  * Recipes
* Navigation must work **without page reload**

---

## Technical Requirements

* Use **functional components**
* Use **useEffect** for API calls
* Use **useState** for managing data
* Use **react-router-dom** for routing
* Use `fetch` for API calls
* Code should be readable and properly structured
