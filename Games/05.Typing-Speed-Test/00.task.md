### 1. **"Lightning Fast!" When WPM > 100**
- **What**: Make **WPM text bold** when it first goes above 100.
- **Where**: In `updateStats()` → check WPM and add `style.fontWeight = 'bold'`.

---

### 2. **"Perfect!" on 100% Accuracy**
- **What**: Turn **accuracy text green** when it hits 100%.
- **Where**: In `updateStats()` → `if (accuracy == 100) style.color = 'green'`.

---

### 3. **"Keep Typing!" When Paused**
- **What**: Show **"Keep typing!"** in blue if user stops typing for 3 seconds.
- **Where**: Use a small `setTimeout` in `input` → reset on each keystroke.

---

### 4. **"First Word!" on First Space**
- **What**: Make **WPM text flash once** (bold → normal) when user types first space.
- **Where**: In `input` event → detect first `' '` and toggle `fontWeight`.

---

### 5. **"New Personal Best!"**
- **What**: Make **Best WPM text red and bold** when a new record is set.
- **Where**: In `saveBestWPM()` → `bestWPMDisplay.style.color = 'red'; style.fontWeight = 'bold'`.

---

### 6. **"Time Running Out!" at 10 Seconds**
- **What**: Make **timer text red and larger** when ≤ 10 seconds.
- **Where**: In `updateTimer()` → `if (timeLeft <= 10) style.color = 'red'; style.fontSize = '2.2em'`.

---

### 7. **"Last Test: 78 WPM" (sessionStorage)**
- **What**: Show **last test’s WPM** under stats — only in this tab.
- **Where**:  
  - Save in `endTest()`: `sessionStorage.setItem('lastWPM', finalWPM)`  
  - Show on load: `sessionStorage.getItem('lastWPM')`  
  - Use simple `<div id="lastTest">Last test: —</div>`

---

### 8. **"Fastest Start" (sessionStorage)**
- **What**: Save **how fast** user typed first 5 characters (in ms).
- **Where**:  
  - On 5th char → `sessionStorage.setItem('fastStart', timeTaken)`  
  - Show: `"Fastest start: 1.2s"` under stats
