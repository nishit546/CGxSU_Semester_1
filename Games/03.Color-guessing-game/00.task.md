### 1. **Correct Color Glows When Clicked**
- **What**: When player clicks the **correct** color, make it **glow** (add a yellow border).
- **Where**: In `handleCorrectGuess()` → add `style.border` to the clicked box.

### 2. **"Streak!" Message When Streak ≥ 3**
- **What**: If `currentStreak` reaches 3 or more, show **"Streak!"** in green.
- **Where**: In `handleCorrectGuess()` → check `currentStreak` and update message.

---

### 3. **Easy Mode Button Turns Green When Selected**
- **What**: When **Easy** mode is active, make the Easy button **green**.
- **Where**: In `setEasyMode()` → add `style.backgroundColor = 'lightgreen'`.

---

### 4. **Show "First Win!" on First Correct Answer**
- **What**: If `currentStreak` goes from 0 to 1, show **"First Win!"** once.
- **Where**: In `handleCorrectGuess()` → check if `currentStreak === 1`.

---

### 5. **Header Text Becomes Bold on New Best Streak**
- **What**: When a **new best streak** is set, make the RGB text **bold**.
- **Where**: In `handleCorrectGuess()` → when saving new best → `colorDisplay.style.fontWeight = 'bold'`.

---

### 6. **Wrong Box Shakes When Clicked**
- **What**: When player picks **wrong**, make that box **shake** (add a CSS class).
- **Where**: In `handleWrongGuess()` → `clickedBox.classList.add('shake')`.
