# **Node.js Interview Questions (Fresher | Practical)**

## **Basic Server & Routing**

1. **Create Basic Express Server**
   Create a server on port 5000 that returns `"Server running"` on `/`

2. **GET API (Static Response)**
   Create `/api/message` that returns JSON `{ message: "Hello World" }`

3. **POST API (Read Body)**
   Create `/api/user` that accepts `{ name }` and returns `"Hello <name>"`

4. **PUT API (Update Data)**
   Update a dummy user object with new name from request body

5. **DELETE API (Simple Response)**
   Create delete route that returns `"User deleted"`

---

## **Middleware (Very Common)**

6. **Logger Middleware**
   Log request method and URL for every request

7. **Time Logger Middleware**
   Log current timestamp for each request

8. **Custom Middleware (Auth Check)**
   Allow request only if query `?token=123` exists

9. **Apply Middleware to Specific Route**
   Apply middleware only on `/admin` route

---

## **Request Handling**

10. **Query Params Handling**
    Create API `/search?name=xyz` and return the name

11. **Route Params Handling**
    Create `/user/:id` and return the ID

12. **Validate Request Body (Manual)**
    If name is missing → return error

13. **Return Proper Status Codes**
    Return 400 for bad request, 200 for success
