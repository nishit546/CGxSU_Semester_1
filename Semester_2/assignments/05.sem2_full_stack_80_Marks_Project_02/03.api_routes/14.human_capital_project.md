# Basic CRUD Routes

| Method | Endpoint | Description |
| :--- | :--- | :--- |
| **GET** | `/prices` | Fetch all price records |
| **GET** | `/prices/:priceId` | Fetch single price record using ID |

| **PUT** | `/prices/:priceId` | Replace complete price record |

| **GET** | `/countries` | Fetch all countries |
| **POST** | `/countries` | Create new country |
| **GET** | `/indicators` | Fetch all indicators |
| **POST** | `/indicators` | Create new indicator |

| **GET** | `/years` | Fetch year records |
