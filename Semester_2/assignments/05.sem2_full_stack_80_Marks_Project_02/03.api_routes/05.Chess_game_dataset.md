# Match Management & CRUD Routes

| Method     | Endpoint                            | Description                        |
| :--------- | :---------------------------------- | :--------------------------------- |
| **GET**    | `/api/v1/matches`                   | Fetch all chess matches            |
| **GET**    | `/api/v1/matches/:matchId`          | Fetch complete match details       |
| **POST**   | `/api/v1/matches`                   | Create a new chess match           |
| **PUT**    | `/api/v1/matches/:matchId`          | Replace complete match information |
| **PATCH**  | `/api/v1/matches/:matchId`          | Partially update match fields      |
| **DELETE** | `/api/v1/matches/:matchId`          | Permanently delete a match         |
| **GET**    | `/api/v1/matches/:matchId/moves`    | Fetch moves of a match             |
| **GET**    | `/api/v1/matches/:matchId/pgn`      | Fetch PGN notation                 |
| **GET**    | `/api/v1/matches/:matchId/fen`      | Fetch final board FEN              |
| **GET**    | `/api/v1/matches/:matchId/analysis` | Fetch engine analysis              |
| **GET**    | `/api/v1/matches/latest`            | Fetch latest matches               |
| **GET**    | `/api/v1/matches/trending`          | Fetch trending matches             |
| **GET**    | `/api/v1/matches/random`            | Fetch random match                 |
| **PATCH**  | `/api/v1/matches/:matchId/archive`  | Archive a match                    |
| **PATCH**  | `/api/v1/matches/:matchId/restore`  | Restore archived match             |
