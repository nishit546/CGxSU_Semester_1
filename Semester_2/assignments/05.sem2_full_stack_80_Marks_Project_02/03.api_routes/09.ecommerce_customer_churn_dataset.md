## Basic CRUD Routes
| Method | Endpoint | Description |
| :--- | :--- | :--- |
| **`GET`** | `/customers` | Fetch all customer records from dataset |
| **`GET`** | `/customers/:id` | Fetch single customer record using ID |
| **`POST`** | `/customers` | Add a new customer record |
| **`PUT`** | `/customers/:id` | Replace complete customer record |
| **`PATCH`** | `/customers/:id` | Update specific customer fields |
| **`DELETE`** | `/customers/:id` | Remove customer record from dataset |
| **`GET`** | `/customers/exists/:id` | Check whether customer exists or not |
| **`POST`** | `/customers/bulk-create` | Insert multiple customer records together |
| **`PATCH`** | `/customers/bulk-update` | Update multiple customer records together |
| **`DELETE`** | `/customers/bulk-delete` | Delete multiple customer records |
