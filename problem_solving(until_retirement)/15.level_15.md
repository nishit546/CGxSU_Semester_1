
## Level 15  – Arrays and logic 

---

### Q181. Sum of Odd Numbers in Array

Return sum of only odd numbers.

* Input: `[1, 2, 3, 4, 5]` → Odds: 1,3,5 → Output: `9`
* Input: `[2, 2, 2]` → Odds: none → Output: `0`
* Input: `[-1, -2, -3]` → Odds: -1,-3 → Output: `-4`

---

### Q182. Swap Elements at Indices i and j

Given an array and two indices `i` and `j`, swap elements at those indices.

* Input: `arr = [1, 2, 3, 4], i = 1, j = 3`
  → Output array: `[1, 4, 3, 2]`

* Input: `arr = ["a", "b", "c"], i = 0, j = 2`
  → Output: `["c", "b", "a"]`

* Input: `arr = [10, 20], i = 0, j = 0`
  → Output: `[10, 20]` (no change)

---

### Q183. Rotate Array Right by k Positions

Right rotation: last elements move to front.

* Input: `arr = [1, 2, 3, 4, 5], k = 1`
  → Output: `[5, 1, 2, 3, 4]`

* Input: `arr = [1, 2, 3, 4, 5], k = 2`
  → Output: `[4, 5, 1, 2, 3]`

* Input: `arr = [1, 2, 3], k = 5`
  k % 3 = 2 → Output: `[2, 3, 1]`

---

### Q184. Rotate Array Left by k Positions

Left rotation: first elements move to end.

* Input: `arr = [1, 2, 3, 4, 5], k = 1`
  → Output: `[2, 3, 4, 5, 1]`

* Input: `arr = [1, 2, 3, 4, 5], k = 2`
  → Output: `[3, 4, 5, 1, 2]`

* Input: `arr = [1, 2, 3], k = 5`
  k % 3 = 2 → Output: `[3, 1, 2]`

---

### Q185. Merge Two Sorted Arrays into One Sorted Array

Both arrays are sorted; merge into a single sorted array (without sorting at the end).

* Input: `[1, 3, 5]` and `[2, 4, 6]`
  → Output: `[1, 2, 3, 4, 5, 6]`

* Input: `[1, 2, 2]` and `[2, 3]`
  → Output: `[1, 2, 2, 2, 3]`

* Input: `[]` and `[1, 2]`
  → Output: `[1, 2]`

### Q186. Bubble Sort (Ascending)

Sort an array using bubble sort algorithm.

* Input: `[3, 1, 4, 2]` → Output: `[1, 2, 3, 4]`
* Input: `[5, 5, 5]` → Output: `[5, 5, 5]`
* Input: `[-1, 3, 0]` → Output: `[-1, 0, 3]`

---

### Q187. Selection Sort (Ascending)

Sort an array using selection sort.

* Input: `[3, 1, 4, 2]` → Output: `[1, 2, 3, 4]`
* Input: `[10, -1, 2]` → Output: `[-1, 2, 10]`
* Input: `[5]` → Output: `[5]`

---

### Q188. Insertion Sort (Ascending)

Sort an array using insertion sort.

* Input: `[3, 1, 4, 2]` → Output: `[1, 2, 3, 4]`
* Input: `[10, -1, 2]` → Output: `[-1, 2, 10]`
* Input: `[5, 4, 3, 2, 1]` → Output: `[1, 2, 3, 4, 5]`

---

### Q189. k-th Smallest Element (Using Sort)

Return the k-th smallest element (1-based index) after sorting.

* Input: `arr = [7, 10, 4, 3, 20, 15], k = 3`
  Sorted: [3,4,7,10,15,20] → 3rd = `7`

* Input: `arr = [1, 2, 3], k = 1` → Output: `1`

* Input: `arr = [5, 5, 5], k = 2` → Sorted: [5,5,5] → 2nd = `5`

---

### Q190. k-th Largest Element (Using Sort)

Return the k-th largest element (1-based index) after sorting.

* Input: `arr = [7, 10, 4, 3, 20, 15], k = 2`
  Sorted descending: [20,15,10,7,4,3] → 2nd = `15`

* Input: `arr = [1, 2, 3], k = 1` → Largest = `3` → Output: `3`

* Input: `arr = [5, 5, 5], k = 3` → Output: `5`
