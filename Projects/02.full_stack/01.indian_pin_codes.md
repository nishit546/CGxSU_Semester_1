# Full Stack Application [Backend APIs + What React Will Do]

- Use Tailwind for React Components
- MongoDB Atlas to Store data
- Node/Express for backend
---

# 1. Get All States

```http
GET /api/states
```

### Response

```json
[
  "GUJARAT",
  "MAHARASHTRA",
  "TAMIL NADU"
]
```

### React Usage

* Populate **State Dropdown** (FilterPanel)
* Used in `/explore` page on load

---

# 2. Get Districts by State

```http
GET /api/states/:state/districts
```

### Example

```http
/api/states/GUJARAT/districts
```

### React Usage

* Trigger when **state is selected**
* Populate **District Dropdown**
* Reset district + taluk when state changes

---

# 3. Get Taluks by District

```http
GET /api/states/:state/districts/:district/taluks
```

### React Usage

* Trigger when **district is selected**
* Populate **Taluk Dropdown**

---

# 4. Get Filtered PIN Code Data (MAIN API)

```http
GET /api/pincodes
```

### Query Params

```http
/api/pincodes?state=GUJARAT&district=AHMEDABAD&taluk=DASKROI&page=1&limit=20
```

### Response

```json
{
  "data": [...],
  "total": 120,
  "page": 1,
  "limit": 20
}
```

### React Usage

* Populate **Data Table**
* Trigger when:

  * Filters change
  * Page changes
* Used in `/explore`

---

# 5. Search API

```http
GET /api/search?q=adi
```

### React Usage

* Used in **SearchBar**
* Debounced input (important)
* Show suggestions OR update table

---

# 6. Get Details by PIN Code

```http
GET /api/pincode/:pincode
```

### Example

```http
/api/pincode/504293
```

### React Usage

* Used in `/pincode` page
* Show result card with full details
