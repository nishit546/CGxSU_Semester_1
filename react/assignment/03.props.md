# React Assignment 2 – Props & Component Communication

## Objective

To understand:

* Passing data using **props**
* Reusing components
* Parent → Child communication
* Rendering dynamic UI using props

---

## Task Description

Build a React application that displays a list of **students** using reusable components.

---

## Requirements

### Components Structure

* `App`
* `StudentList`
* `StudentCard`

---

### Student Data

Create an array of students in `App.js`:

* Name
* Age
* Course
* City

---

### Functional Requirements

* `App` passes the student list to `StudentList` using props
* `StudentList` passes individual student data to `StudentCard`
* `StudentCard` displays:

  * Student Name
  * Age
  * Course
  * City

---

### Rules

* Do not hardcode values inside `StudentCard`
* All data must come from props
* Use `.map()` to render the list

---

## Evaluation

* Correct use of props
* Component reusability
* Clean and readable JSX

---
