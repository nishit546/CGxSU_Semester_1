# Animations in React Native

In React Native, **animations** bring your app to life. They make interfaces feel responsive, polished, and delightful. Mastering animations is key to creating professional-grade mobile apps.

---

## What are Animations?

Animations improve **User Experience (UX)** by providing visual feedback, guiding attention, and making interactions feel natural and engaging.

**Real-life Analogy**:  
Just like how a door smoothly opens instead of instantly appearing, or how a button physically presses down, animations make digital interactions feel intuitive and satisfying.

### Common Use Cases
- Button press feedback (scale, opacity, color change)
- Screen transitions and navigation animations
- Loading states and progress indicators
- Swipe gestures (like Tinder cards or swipe-to-delete)
- Collapsible menus, bottom sheets, and modals
- Scroll-driven animations and parallax effects
- Empty states and success celebrations
- Micro-interactions that delight users

---

### **Simple Example (Animated API - Fade In)**

```jsx
import React, { useRef, useEffect } from 'react';
import { Animated, View, Text, StyleSheet, Button } from 'react-native';

const App = () => {
  const fadeAnim = useRef(new Animated.Value(0)).current;

  useEffect(() => {
    Animated.timing(fadeAnim, {
      toValue: 1,
      duration: 1200,
      useNativeDriver: true,
    }).start();
  }, [fadeAnim]);

  return (
    <View style={styles.container}>
      <Animated.View style={[styles.card, { opacity: fadeAnim }]}>
        <Text style={styles.title}>Welcome to Animations!</Text>
        <Text style={styles.subtitle}>This card faded in smoothly.</Text>
      </Animated.View>
      <Button title="Fade Again" onPress={() => {
        fadeAnim.setValue(0);
        Animated.timing(fadeAnim, {
          toValue: 1,
          duration: 800,
          useNativeDriver: true,
        }).start();
      }} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#f8f9fa',
  },
  card: {
    backgroundColor: 'white',
    padding: 30,
    borderRadius: 20,
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 4 },
    shadowOpacity: 0.1,
    shadowRadius: 10,
    elevation: 8,
    alignItems: 'center',
    marginBottom: 30,
  },
  title: {
    fontSize: 24,
    fontWeight: 'bold',
    marginBottom: 8,
  },
  subtitle: {
    fontSize: 16,
    color: '#666',
    textAlign: 'center',
  },
});

export default App;
```

### **Types of Animation Approaches**

| Type                  | Description                                      | Best For |
|-----------------------|--------------------------------------------------|----------|
| **Animated API**      | Built-in React Native animation library         | Simple transitions, fades, springs |
| **Reanimated**        | High-performance library (UI thread)            | Complex gestures, scroll animations |
| **Gesture Handler**   | Advanced touch & gesture recognition            | Swipe, pan, pinch interactions |
| **Lottie**            | JSON-based vector animations                    | Complex illustrations, loading, success |

## 1. Animated API

**Description**: `Animated` is React Native’s built-in library for creating smooth animations. It works directly with React Native components.

### **What is Animated API?**
It allows you to animate numeric values (opacity, scale, translate, etc.) over time using timing, spring, or decay functions. Almost every animated UI effect starts with `Animated.Value`.

---

### **Basic Complete Example**

```jsx
import React, { useRef } from 'react';
import { Animated, View, Text, StyleSheet, TouchableOpacity } from 'react-native';

const App = () => {
  const scaleAnim = useRef(new Animated.Value(1)).current;
  const rotateAnim = useRef(new Animated.Value(0)).current;

  const animateButton = () => {
    Animated.parallel([
      Animated.spring(scaleAnim, {
        toValue: 0.9,
        friction: 3,
        useNativeDriver: true,
      }),
      Animated.timing(rotateAnim, {
        toValue: 1,
        duration: 300,
        useNativeDriver: true,
      }),
    ]).start(() => {
      Animated.spring(scaleAnim, {
        toValue: 1,
        friction: 3,
        useNativeDriver: true,
      }).start();
      rotateAnim.setValue(0);
    });
  };

  const rotateInterpolate = rotateAnim.interpolate({
    inputRange: [0, 1],
    outputRange: ['0deg', '360deg'],
  });

  return (
    <View style={styles.container}>
      <Animated.View
        style={[
          styles.button,
          {
            transform: [
              { scale: scaleAnim },
              { rotate: rotateInterpolate },
            ],
          },
        ]}
      >
        <TouchableOpacity onPress={animateButton} activeOpacity={1}>
          <Text style={styles.buttonText}>Press Me!</Text>
        </TouchableOpacity>
      </Animated.View>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#fff',
  },
  button: {
    backgroundColor: '#007AFF',
    paddingVertical: 20,
    paddingHorizontal: 50,
    borderRadius: 15,
  },
  buttonText: {
    color: 'white',
    fontSize: 18,
    fontWeight: 'bold',
  },
});

export default App;
```

---

### **Important Concepts**

| Concept              | What it does                                      | Common Values |
|----------------------|---------------------------------------------------|---------------|
| `Animated.Value`     | Tracks animated numeric value                     | `new Animated.Value(0)` |
| `timing()`           | Animation with fixed duration                     | `{duration: 500}` |
| `spring()`           | Natural bouncy animation                          | `{friction: 5}` |
| `useNativeDriver`    | Runs animation on native thread (performance)     | `true` |
| `interpolate()`      | Maps input range to output range                  | `inputRange`, `outputRange` |

---

### **More Examples**

#### **1. Slide In from Bottom**
```jsx
const translateY = useRef(new Animated.Value(100)).current;

Animated.timing(translateY, {
  toValue: 0,
  duration: 600,
  useNativeDriver: true,
}).start();

<Animated.View style={{ transform: [{ translateY }] }}>
  {/* Content */}
</Animated.View>
```

#### **2. Sequence of Animations**
```jsx
Animated.sequence([
  Animated.timing(fadeAnim, { toValue: 1, useNativeDriver: true }),
  Animated.spring(scaleAnim, { toValue: 1.1, useNativeDriver: true }),
]).start();
```

#### **3. Looping Animation**
```jsx
Animated.loop(
  Animated.sequence([
    Animated.timing(anim, { toValue: 1, duration: 800, useNativeDriver: true }),
    Animated.timing(anim, { toValue: 0, duration: 800, useNativeDriver: true }),
  ])
).start();
```

## 2. Reanimated

**Description**: Reanimated is a powerful library that runs animations on the UI thread for superior performance.

### **What is Reanimated?**
It provides a more modern and performant way to create complex animations and gesture-driven interactions. It works seamlessly with Gesture Handler.

### **Basic Complete Example**

```jsx
import React from 'react';
import { View, Button, StyleSheet } from 'react-native';
import Animated, {
  useSharedValue,
  useAnimatedStyle,
  withSpring,
  withTiming,
} from 'react-native-reanimated';

const App = () => {
  const offset = useSharedValue(0);
  const opacity = useSharedValue(1);

  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ translateX: offset.value }],
    opacity: opacity.value,
  }));

  const triggerAnimation = () => {
    offset.value = withSpring(250);
    opacity.value = withTiming(0.3, { duration: 400 });
    
    setTimeout(() => {
      offset.value = withSpring(0);
      opacity.value = withTiming(1);
    }, 1200);
  };

  return (
    <View style={styles.container}>
      <Animated.View style={[styles.box, animatedStyle]} />
      <Button title="Animate Box" onPress={triggerAnimation} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: { flex: 1, justifyContent: 'center', alignItems: 'center' },
  box: {
    width: 180,
    height: 180,
    backgroundColor: '#FF2D55',
    borderRadius: 25,
    marginBottom: 40,
  },
});

export default App;
```

---

### **Important Concepts (Reanimated)**

| Concept                | What it does                                      | Example |
|------------------------|---------------------------------------------------|---------|
| `useSharedValue`       | Mutable value shared with UI thread               | `useSharedValue(0)` |
| `useAnimatedStyle`     | Creates reactive animated styles                  | Returns style object |
| `withTiming`           | Duration-based animation                          | `withTiming(100)` |
| `withSpring`           | Physics-based natural animation                   | `withSpring(200)` |

---

### **More Examples**

#### **Gesture Controlled Drag (Combined with Gesture Handler)**
```jsx
// (See Gesture Handler section below for full integration)
const translationX = useSharedValue(0);

const animatedStyle = useAnimatedStyle(() => ({
  transform: [{ translateX: translationX.value }],
}));
```

## 3. Gesture Handler + Reanimated

**Description**: Enables advanced touch gestures like pan, pinch, tap, and long press.

### **Basic Complete Example (Swipeable Card)**

```jsx
import React from 'react';
import { View, StyleSheet } from 'react-native';
import { Gesture, GestureDetector } from 'react-native-gesture-handler';
import Animated, { useSharedValue, useAnimatedStyle, withSpring } from 'react-native-reanimated';

const App = () => {
  const translateX = useSharedValue(0);

  const panGesture = Gesture.Pan()
    .onUpdate((event) => {
      translateX.value = event.translationX;
    })
    .onEnd((event) => {
      if (Math.abs(event.translationX) > 150) {
        translateX.value = withSpring(event.translationX > 0 ? 400 : -400);
      } else {
        translateX.value = withSpring(0);
      }
    });

  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ translateX: translateX.value }],
  }));

  return (
    <View style={styles.container}>
      <GestureDetector gesture={panGesture}>
        <Animated.View style={[styles.card, animatedStyle]}>
          <Text style={styles.cardText}>Swipe Me!</Text>
        </Animated.View>
      </GestureDetector>
    </View>
  );
};

const styles = StyleSheet.create({
  container: { flex: 1, justifyContent: 'center', alignItems: 'center' },
  card: {
    width: 300,
    height: 200,
    backgroundColor: '#007AFF',
    borderRadius: 20,
    justifyContent: 'center',
    alignItems: 'center',
  },
  cardText: { color: 'white', fontSize: 22, fontWeight: 'bold' },
});

export default App;
```

## 4. Lottie Animations

**Description**: Lottie plays high-quality JSON animations exported from After Effects.

### **Basic Complete Example**

```jsx
import React, { useRef } from 'react';
import { View, StyleSheet, Button } from 'react-native';
import LottieView from 'lottie-react-native';

const App = () => {
  const animation = useRef(null);

  return (
    <View style={styles.container}>
      <LottieView
        ref={animation}
        source={require('./assets/success.json')} // Add your .json file
        style={styles.lottie}
        autoPlay={false}
        loop={false}
      />
      <Button
        title="Play Animation"
        onPress={() => animation.current?.play()}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: { flex: 1, justifyContent: 'center', alignItems: 'center' },
  lottie: { width: 250, height: 250 },
});

export default App;
```

---

### **More Examples**

- **Loading Animation**: `autoPlay loop={true}`
- **One-time Success**: `loop={false}` + `onAnimationFinish`
- **Progress Control**: `progress={progressValue}`

## Choosing the Right Tool

| Requirement                     | Recommended Tool              |
|---------------------------------|-------------------------------|
| Simple fade/slide/button effect | Animated API                  |
| Complex gestures + smooth UI    | Reanimated + Gesture Handler  |
| Beautiful designer animations   | Lottie                        |
| Scroll / list animations        | Reanimated                    |

---

## Performance Tips & Best Practices

- Always use `useNativeDriver: true` with Animated API
- Prefer Reanimated for anything beyond simple animations
- Keep animation logic lightweight
- Test thoroughly on real devices
- Respect `AccessibilityInfo.reduceMotion()`
- Keep durations short (200-500ms typically)
