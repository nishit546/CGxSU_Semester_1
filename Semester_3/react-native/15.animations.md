# 2️⃣ Animated API

The Animated API is React Native's built-in animation library.

## Create an Animated Value

```jsx
import { Animated } from "react-native";

const fadeAnim = new Animated.Value(0);
```

## Fade In Animation

```jsx
Animated.timing(fadeAnim, {
  toValue: 1,
  duration: 1000,
  useNativeDriver: true,
}).start();
```

Apply animation:

```jsx
<Animated.View style={{ opacity: fadeAnim }} />
```

### Animation Types

- timing()
- spring()
- decay()

# 3️⃣ Reanimated

Reanimated runs animations on the UI thread for smoother performance.

## Installation

```bash
npx expo install react-native-reanimated
```

## Shared Value

```jsx
const offset = useSharedValue(0);
```

## Animated Style

```jsx
const style = useAnimatedStyle(() => ({
  transform: [{ translateX: offset.value }],
}));
```

## Trigger Animation

```jsx
offset.value = withTiming(200);
```

### Why Reanimated?

- Better performance
- 60 FPS animations
- Complex gesture support
- Production-ready

# 4️⃣ Gesture Handler

Gesture Handler enables advanced touch interactions.

## Installation

```bash
npx expo install react-native-gesture-handler
```

### Common Gestures

- Tap
- Double Tap
- Long Press
- Pan
- Pinch
- Rotation

Example:

```jsx
<GestureDetector gesture={tapGesture}>
  <Animated.View />
</GestureDetector>
```

---

# 5️⃣ Lottie Animations

Lottie plays JSON-based animations created in Adobe After Effects.

## Installation

```bash
npx expo install lottie-react-native
```

## Example

```jsx
import LottieView from "lottie-react-native";

<LottieView
  source={require("./assets/loading.json")}
  autoPlay
  loop
/>
```

### Use Cases

- Splash Screen
- Success Animation
- Loading Screen
- Empty State
- Payment Success

# 6️⃣ Choosing the Right Animation Tool

| Requirement | Recommended Tool |
|-------------|------------------|
| Simple Fade | Animated API |
| Complex UI Animation | Reanimated |
| Swipe Gestures | Gesture Handler |
| JSON Animation | Lottie |

# 7️⃣ Performance Tips

- Prefer Reanimated for complex animations
- Use `useNativeDriver` where supported
- Avoid unnecessary re-renders
- Keep animation durations reasonable
- Test on real devices

---

#  Best Practices

- Keep animations meaningful
- Maintain consistent animation timing
- Provide feedback for user actions
- Avoid excessive animations
- Respect accessibility settings

---

#  Common Mistakes

- Running heavy work during animations
- Overusing animations
- Forgetting `useNativeDriver`
- Using Lottie for everything
- Not cleaning up gesture listeners
