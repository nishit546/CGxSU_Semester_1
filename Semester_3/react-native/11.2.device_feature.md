# 1️⃣ Understanding Permissions

Many device features require user permission.

Examples:

- Camera
- Gallery
- Location
- Contacts
- Microphone

Permission Flow

```text
App Starts
    │
Request Permission
    │
Granted?
 ├── Yes → Access Feature
 └── No  → Show Permission Screen
```

### Best Practices

- Request permission only when needed.
- Explain why permission is required.
- Handle denied permissions gracefully.

# 2️⃣ Location (`expo-location`)

## What is Location?

Location allows your app to access the user's geographical position using GPS, Wi-Fi, or cellular networks.

### Real-world Use Cases

- Food Delivery
- Ride Booking
- Attendance
- Weather Apps
- Navigation

---

## Installation

```bash
npx expo install expo-location
```

---

## Request Location Permission

```jsx
import * as Location from "expo-location";

const { status } =
await Location.requestForegroundPermissionsAsync();

if (status !== "granted") {
  return;
}
```

---

## Get Current Location

```jsx
const location =
await Location.getCurrentPositionAsync({});

console.log(location.coords.latitude);
console.log(location.coords.longitude);
```

Returned data:

- latitude
- longitude
- altitude
- accuracy
- speed

---

## Watch User Location (Live Tracking)

```jsx
const subscription =
await Location.watchPositionAsync(
  {
    accuracy: Location.Accuracy.High,
    distanceInterval: 10,
  },
  (location) => {
    console.log(location.coords);
  }
);
```

Stop tracking:

```jsx
subscription.remove();
```

---

## Reverse Geocoding

Convert coordinates into an address.

```jsx
const address =
await Location.reverseGeocodeAsync({
  latitude,
  longitude,
});
```

# 3️⃣ Maps (`react-native-maps`)

## Installation

```bash
npx expo install react-native-maps
```

---

## Basic Map

```jsx
import MapView from "react-native-maps";

<MapView
  style={{ flex: 1 }}
/>
```

---

## Initial Region

```jsx
<MapView
  initialRegion={{
    latitude: 28.6139,
    longitude: 77.2090,
    latitudeDelta: 0.05,
    longitudeDelta: 0.05,
  }}
/>
```

---

## Add Marker

```jsx
import { Marker } from "react-native-maps";

<Marker
  coordinate={{
    latitude: 28.6139,
    longitude: 77.2090,
  }}
  title="Current Location"
/>
```

---

## Map Use Cases

- Delivery Tracking
- Cab Booking
- Nearby Stores
- Travel Apps

# 4️⃣ Contacts (`expo-contacts`)

## Installation

```bash
npx expo install expo-contacts
```

---

## Permission

```jsx
import * as Contacts from "expo-contacts";

const { status } =
await Contacts.requestPermissionsAsync();
```

---

## Read Contacts

```jsx
const { data } =
await Contacts.getContactsAsync();

console.log(data);
```

Each contact may contain:

- Name
- Phone Number
- Email
- Company

# 5️⃣ Live Tracking Architecture

```text
GPS
   │
expo-location
   │
Latitude & Longitude
   │
MapView
   │
Marker Updates
   │
User sees live movement
```

---

#  Best Practices

- Request permissions only when needed.
- Stop location tracking when leaving the screen.
- Use high accuracy only when required.
- Cache location if frequent updates aren't needed.
- Handle permission denial with clear messages.

---

#  Common Mistakes

- Forgetting to remove location subscriptions
- Requesting all permissions on app launch
- Not checking permission status
- Updating the map too frequently
- Ignoring battery usage
