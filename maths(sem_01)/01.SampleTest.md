# SAMPLE QUESTION PAPER



## Q.1 — MCQs (12 × 1 = 12 Marks)

1. **A well-defined collection of distinct objects is called a:**  
   A. Relation  
   B. Set  
   C. Function  
   D. Graph  

2. **Two sets A and B are equal if:**  
   A. They have the same number of elements  
   B. A ⊆ B  
   C. They contain exactly the same elements  
   D. Their power sets are different  

3. **The number of elements in the power set of a set with 5 elements is:**  
   A. 5  
   B. 10  
   C. 25  
   D. 32  

4. **A simple graph does NOT contain:**  
   A. Vertices  
   B. Edges  
   C. Self-loops  
   D. Undirected edges  

5. **In BFS traversal, the first vertex visited is:**  
   A. The last vertex  
   B. The vertex with highest degree  
   C. The starting vertex  
   D. Any random vertex  

6. **A tree with 6 vertices has how many edges?**  
   A. 4  
   B. 5  
   C. 6  
   D. 7  

7. **The probability of an impossible event is:**  
   A. 1  
   B. 0  
   C. −1  
   D. 0.5  

8. **If events A and B are mutually exclusive, then:**  
   A. P(A ∩ B) = 1  
   B. P(A ∩ B) = 0  
   C. P(A) = P(B)  
   D. P(A | B) = 1  

9. **The median of a data set is the:**  
   A. Most frequent value  
   B. Average of values  
   C. Middle value after arranging data  
   D. Largest value  

10. **A matrix with all elements equal to zero is called:**  
    A. Identity matrix  
    B. Square matrix  
    C. Zero matrix  
    D. Diagonal matrix  

11. **The transpose of a matrix changes:**  
    A. Its determinant  
    B. Rows into columns  
    C. Its rank  
    D. Its order always  

12. **Vectors are linearly independent if:**  
    A. One vector is a multiple of another  
    B. Only the trivial solution exists  
    C. Their sum is zero  
    D. They are equal
