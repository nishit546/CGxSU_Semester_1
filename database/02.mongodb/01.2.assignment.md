# **MongoDB Setup – Simple Assignment (Atlas + Compass)**

The goal of this assignment is to make sure you can **connect a cloud MongoDB database using MongoDB Compass**.

---

## **Assignment Objective**

By the end of this assignment, you should be able to:

* Create a MongoDB Atlas cluster
* Connect it to MongoDB Compass
* Create a database and collection using the UI

---

## **Task 1: Create MongoDB Atlas Cluster**

1. Log in to MongoDB Atlas.
2. Create a **Free Tier (M0)** cluster.
3. Wait until the cluster status shows **Running**.

---

## **Task 2: Configure Access**

1. Create **one database user** (username + password).
2. Add your **IP address** in Network Access
   *(Allow from anywhere is fine for learning)*.

---

## **Task 3: Install MongoDB Compass**

1. Download MongoDB Compass from the official website.
2. Install and open MongoDB Compass.

---

## **Task 4: Connect Atlas to Compass**

1. In Atlas, click **Connect → MongoDB Compass**.
2. Copy the connection string.
3. Replace `<username>` and `<password>`.
4. Paste it into MongoDB Compass and click **Connect**.

You should now see your cluster connected in Compass.

---

## **Task 5: Create Database & Collection**

Using MongoDB Compass:

1. Create a database named:
   `practice_db`
2. Create a collection named:
   `demo_collection`
3. Insert **one sample document** using the UI.

---

## **Completion Criteria**

You are done when:

* Atlas is connected to Compass
* You can create a database and collection
