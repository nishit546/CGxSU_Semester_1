# 1️⃣ What is React Native?

React Native is an **open-source JavaScript framework** developed by **Meta (Facebook)** to build **native mobile applications** for Android and iOS using **React**.

Instead of writing separate Android and iOS applications, React Native allows developers to build both using **one JavaScript codebase**.

### Example

React.js

```jsx
<div>Hello</div>
```

React Native

```jsx
<Text>Hello</Text>
```

Unlike React.js, React Native **does not render HTML**. It renders **real native components**.

# 4️⃣ React Native Architecture

## Old Architecture

```
React Code
      │
JavaScript Thread
      │
    Bridge
      │
Native Thread
      │
Android / iOS UI
```

### JavaScript Thread

Runs your React code.

### Bridge

Transfers messages between JavaScript and Native.

### Native Thread

Renders actual Android and iOS components.

---

## New Architecture

React Native introduced a new architecture to improve performance.

It consists of:

- JSI (JavaScript Interface)
- Fabric Renderer
- Turbo Modules

### JSI

Allows JavaScript to communicate directly with Native code without relying on the old Bridge.

### Fabric

The new rendering engine that makes UI updates faster.

### Turbo Modules

Loads Native Modules only when required, improving startup performance.

# 6️⃣ React.js vs React Native

| React.js | React Native |
|----------|--------------|
| Web Applications | Mobile Applications |
| HTML Elements | Native Components |
| CSS | StyleSheet |
| Browser DOM | Native UI |
| React Router | React Navigation |

---

# 7️⃣ React Native vs Flutter

| React Native | Flutter |
|--------------|----------|
| JavaScript | Dart |
| Native Components | Own Rendering Engine |
| Easier for React Developers | New Language Required |
| Large Community | Growing Community |

### Which Should You Learn?

Choose React Native if:

- You already know React.js.
- You want faster development.
- You want to reuse JavaScript knowledge.

# 8️⃣ Expo vs React Native CLI

## Expo

### Advantages

- Beginner Friendly
- Fast Setup
- Expo Go Support
- No Native Setup

### Limitations

- Limited Native Customization

---

## React Native CLI

### Advantages

- Full Native Access
- Better for Large Production Apps

### Limitations

- Android Studio and Xcode setup required

# 9️⃣ Creating Your First Expo App

```bash
npx create-expo-app@latest myApp
```

Move inside project:

```bash
cd myApp
```

Run project:

```bash
npx expo start
```

Open **Expo Go** on your phone and scan the QR code.

# 🔟 React Native Folder Structure

```
myApp/

├── app/
├── assets/
├── components/
├── hooks/
├── services/
├── package.json
├── app.json
└── node_modules/
```

### Important Files

| File | Purpose |
|------|---------|
| app | Screens & Routes |
| assets | Images, Fonts |
| components | Reusable UI |
| hooks | Custom Hooks |
| services | API Calls |
| package.json | Dependencies |
| app.json | Expo Configuration |
