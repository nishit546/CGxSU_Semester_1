
# React Native - Core Components in React Native

In React Native, **everything you see on the screen is a Component**. Understanding core components is essential as they are the building blocks of every mobile app.

---

## What are Components?

A **component** is a reusable piece of UI. Just like LEGO blocks, you combine small components to build complex screens.

**Examples**: Login Screen, Header, Button, Card, Profile Page, etc.

---

## 1. View

**Description**: `View` is the most fundamental component. It acts as a container for other components. It is similar to `<div>` in web development.

**Common Props**: `style`, `onLayout`, `accessible`

### Examples:

```jsx
// Example 1: Basic Container
<View style={{ flex: 1, backgroundColor: '#fff' }}></View>

// Example 2: Card Layout
<View style={styles.card}>
  <Text>Product Title</Text>
  <Text>$99.99</Text>
</View>

// Example 3: Centered Content
<View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
  <Text>Welcome</Text>
</View>

// Example 4: Nested Views
<View>
  <View style={styles.row}>
    <Text>Left</Text>
    <Text>Right</Text>
  </View>
</View>
```

---

## 2. Text

**Description**: Used to display text. In React Native, all text must be wrapped inside `<Text>` component.

**Common Props**: `style`, `numberOfLines`, `ellipsizeMode`

### Examples:

```jsx
// Example 1: Simple Text
<Text>Hello React Native!</Text>

// Example 2: Styled Text
<Text style={{ fontSize: 24, fontWeight: 'bold', color: 'blue' }}>
  Welcome to My App
</Text>

// Example 3: Multi-line Text
<Text numberOfLines={2}>
  This is a long text that will be truncated after two lines...
</Text>

// Example 4: Nested Text
<Text>
  Hello <Text style={{ fontWeight: 'bold' }}>World</Text>!
</Text>
```

---

## 3. Image

**Description**: Displays images from local assets or remote URLs.

**Important**: Must specify `width` and `height` for remote images.

### Examples:

```jsx
// Example 1: Local Image
<Image source={require('./assets/logo.png')} style={styles.image} />

// Example 2: Remote Image
<Image
  source={{ uri: 'https://picsum.photos/300' }}
  style={{ width: 300, height: 300, borderRadius: 15 }}
/>

// Example 3: Network Image with Placeholder
<Image
  source={{ uri: 'https://example.com/photo.jpg' }}
  defaultSource={require('./assets/placeholder.png')}
/>

// Example 4: Avatar Style
<Image
  source={{ uri: user.profilePic }}
  style={{ width: 60, height: 60, borderRadius: 30 }}
/>
```

---

## 4. ScrollView

**Description**: Makes content scrollable. Best for small lists or static content.

### Examples:

```jsx
// Example 1: Basic Scroll
<ScrollView>
  <Text>Item 1</Text>
  <Text>Item 2</Text>
  <Text>Item 3</Text>
</ScrollView>

// Example 2: Horizontal Scroll
<ScrollView horizontal>
  <Image source={...} />
  <Image source={...} />
</ScrollView>

// Example 3: With Refresh Control
<ScrollView refreshControl={<RefreshControl refreshing={false} onRefresh={onRefresh} />}>
  {/* Content */}
</ScrollView>
```

---

## 5. FlatList

**Description**: High-performance component for rendering long lists. Only renders visible items (virtualized).

### Examples:

```jsx
const users = [
  { id: '1', name: 'Rahul' },
  { id: '2', name: 'Priya' },
];

// Example 1: Basic FlatList
<FlatList
  data={users}
  keyExtractor={item => item.id}
  renderItem={({ item }) => <Text>{item.name}</Text>}
/>

// Example 2: With Separator
<FlatList
  data={users}
  renderItem={({ item }) => <UserCard user={item} />}
  ItemSeparatorComponent={() => <View style={{ height: 10 }} />}
/>
```

---

## 6. SectionList

**Description**: Used for grouped (sectioned) lists like contacts or settings.

### Examples:

```jsx
const sections = [
  { title: 'A', data: ['Aman', 'Ankit'] },
  { title: 'B', data: ['Bharat', 'Bobby'] }
];

<SectionList
  sections={sections}
  keyExtractor={(item, index) => item + index}
  renderItem={({ item }) => <Text style={styles.item}>{item}</Text>}
  renderSectionHeader={({ section }) => (
    <Text style={styles.header}>{section.title}</Text>
  )}
/>
```

---

## 7. Pressable

**Description**: Modern and highly customizable component for handling touch events.

### Examples:

```jsx
// Example 1: Basic
<Pressable onPress={() => alert('Pressed!')}>
  <Text>Click Me</Text>
</Pressable>

// Example 2: With Style Change
<Pressable 
  onPress={handleLogin}
  style={({ pressed }) => [
    styles.button,
    { opacity: pressed ? 0.7 : 1 }
  ]}
>
  <Text>Login</Text>
</Pressable>
```

---

## 8. TouchableOpacity

**Description**: Classic touchable component that reduces opacity on press.

### Examples:

```jsx
<TouchableOpacity onPress={onSubmit} style={styles.btn}>
  <Text style={styles.btnText}>Submit</Text>
</TouchableOpacity>
```

---

## 9. SafeAreaView

**Description**: Automatically adds padding to avoid notches, status bar, and home indicators.

```jsx
<SafeAreaView style={{ flex: 1 }}>
  <Text>Safe Content Here</Text>
</SafeAreaView>
```

---

## 10. StatusBar

**Description**: Controls the appearance of the device status bar.

```jsx
<StatusBar barStyle="dark-content" backgroundColor="#fff" />
```

---

## 11. Modal

**Description**: Displays content over the main screen.

```jsx
<Modal visible={modalVisible} transparent animationType="slide">
  <View style={styles.modal}>
    <Text>Modal Content</Text>
  </View>
</Modal>
```

---

## 12. ActivityIndicator

**Description**: Shows a loading spinner.

```jsx
<ActivityIndicator size="large" color="#007AFF" />
```

---

## 13. Switch

**Description**: Toggle switch for ON/OFF states.

```jsx
<Switch value={isEnabled} onValueChange={setIsEnabled} />
```

---

## 14. RefreshControl

**Description**: Adds pull-to-refresh functionality.

```jsx
<FlatList
  refreshControl={
    <RefreshControl refreshing={refreshing} onRefresh={onRefresh} />
  }
/>
```
