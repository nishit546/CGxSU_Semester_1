
# React Native - Core Components 

In React Native, **everything you see on the screen is a Component**. Understanding core components is essential as they are the building blocks of every mobile app.

---

## What are Components?

In React Native, a **component** is a self-contained, reusable building block of your app’s user interface. 

Think of components as **small, independent parts** that you can use multiple times across your app. They can be as small as a button or as big as an entire screen.

**Real-life Analogy**:  
Just like how a car is made of smaller parts (wheels, engine, seats), a mobile app screen is made of smaller components (Header, Button, Card, Input Field, etc.).

---

### **Simple Example**

```jsx
import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

// Custom Component
const Button = ({ title, onPress }) => {
  return (
    <View style={styles.button}>
      <Text style={styles.buttonText}>{title}</Text>
    </View>
  );
};

const App = () => {
  return (
    <View style={styles.container}>
      <Text style={styles.heading}>Welcome</Text>
      
      {/* Reusing the same Button component */}
      <Button title="Login" onPress={() => alert('Login Pressed')} />
      <Button title="Sign Up" onPress={() => alert('Sign Up Pressed')} />
      <Button title="Forgot Password?" onPress={() => alert('Forgot Password')} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#fff',
  },
  heading: {
    fontSize: 28,
    fontWeight: 'bold',
    marginBottom: 40,
  },
  button: {
    backgroundColor: '#007AFF',
    paddingVertical: 14,
    paddingHorizontal: 40,
    borderRadius: 10,
    marginVertical: 8,
    width: '80%',
  },
  buttonText: {
    color: 'white',
    fontSize: 16,
    textAlign: 'center',
    fontWeight: 'bold',
  },
});

export default App;
```

---

### **Types of Components**

| Type                  | Description                                      | Example |
|-----------------------|--------------------------------------------------|--------|
| **Built-in Components** | Provided by React Native                        | `View`, `Text`, `Image`, `FlatList`, `Button` |
| **Custom Components**   | Created by you                                   | `Header`, `Card`, `LoginForm`, `ProfilePic` |
| **Third-party Components** | From libraries                              | `react-native-vector-icons`, `lottie-react-native` |

---

### **Why Use Components? (Benefits)**

1. **Reusability** — Write once, use anywhere.
2. **Maintainability** — Easier to update and fix bugs.
3. **Organization** — Clean and structured code.
4. **Collaboration** — Team members can work on different components.
5. **Consistency** — Same button style across the entire app.

## 1. View

**Description**: `View` is the most fundamental component. It acts as a container for other components. It is similar to `<div>` in web development.

### **What is View?**
`View` is the basic building block of React Native UI. Almost every screen you build starts with one or more `View` components. It is used to:
- Group components together
- Apply styles (background, padding, margins, flexbox, etc.)
- Create layouts (cards, rows, columns, containers)

You can think of it as the **main container** for your UI elements.

---

### **Basic Complete Example**

```jsx
import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <View style={styles.card}>
        <Text style={styles.title}>Product Title</Text>
        <Text style={styles.price}>$99.99</Text>
        <Text style={styles.description}>
          This is a beautiful product description.
        </Text>
      </View>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#f5f5f5',
    padding: 20,
    justifyContent: 'center',
  },
  card: {
    backgroundColor: 'white',
    padding: 20,
    borderRadius: 15,
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 2 },
    shadowOpacity: 0.1,
    shadowRadius: 8,
    elevation: 5,           // For Android shadow
  },
  title: {
    fontSize: 22,
    fontWeight: 'bold',
  },
  price: {
    fontSize: 18,
    color: '#007AFF',
    marginVertical: 8,
  },
  description: {
    fontSize: 16,
    color: '#666',
  },
});

export default App;
```

---

### **Important Props**

| Prop           | What it does                                      | Common Values |
|----------------|---------------------------------------------------|---------------|
| `style`        | Main way to style the View                        | Style object |
| `onLayout`     | Get dimensions of the View                        | `({nativeEvent}) => {}` |
| `accessible`   | Makes View accessible for screen readers          | `true` / `false` |

---

### **More Examples**

#### **1. Basic Container**

```jsx
<View style={{ flex: 1, backgroundColor: '#fff' }}>
  {/* Your content */}
</View>
```

#### **2. Centered Content**

```jsx
<View style={{ 
  flex: 1, 
  justifyContent: 'center', 
  alignItems: 'center' 
}}>
  <Text>Welcome to My App</Text>
</View>
```

#### **3. Row Layout (Flex Direction)**

```jsx
<View style={styles.row}>
  <Text>Left Side</Text>
  <Text>Right Side</Text>
</View>
```

**Style:**

```jsx
row: {
  flexDirection: 'row',
  justifyContent: 'space-between',
  padding: 10,
},
```

#### **4. Nested Views**

```jsx
<View style={styles.card}>
  <View style={styles.header}>
    <Text>Header</Text>
  </View>
  <View style={styles.body}>
    <Text>Main Content</Text>
  </View>
</View>
```

## 2. Text

**Description**: Used to display text. In React Native, **all text must be wrapped inside a `<Text>` component**.

### **What is Text?**
The `Text` component is one of the most basic and frequently used components in React Native. Unlike web HTML where you can use `<p>`, `<span>`, `<h1>`, etc., in React Native **everything text-related must be inside `<Text>`**.

It supports styling, nesting, truncation, and more.

---

### **Basic Complete Example**

```jsx
import React from 'react';
import { Text, View, StyleSheet } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <Text style={styles.title}>Hello React Native!</Text>
      
      <Text style={styles.subtitle}>
        This is a simple Text component example.
      </Text>

      <Text style={styles.description}>
        You can style text using fontSize, color, fontWeight, and many other properties.
      </Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    padding: 20,
  },
  title: {
    fontSize: 28,
    fontWeight: 'bold',
    marginBottom: 10,
  },
  subtitle: {
    fontSize: 20,
    color: '#666',
    marginBottom: 15,
  },
  description: {
    fontSize: 16,
    textAlign: 'center',
    lineHeight: 24,
  },
});

export default App;
```

---

### **Important Props (For Beginners)**

| Prop                | What it does                                      | Common Values |
|---------------------|---------------------------------------------------|---------------|
| `style`             | Applies styling (fontSize, color, etc.)           | Style object |
| `numberOfLines`     | Limits the number of lines                        | `1`, `2`, `3` |
| `ellipsizeMode`     | How to truncate text                              | `"tail"`, `"head"`, `"middle"`, `"clip"` |
| `onPress`           | Makes text clickable                              | `() => {}` |

---

### **More Examples**

#### **1. Simple Text**

```jsx
<Text>Hello React Native!</Text>
```

#### **2. Styled Text**

```jsx
<Text style={{ 
  fontSize: 24, 
  fontWeight: 'bold', 
  color: 'blue',
  textAlign: 'center'
}}>
  Welcome to My App
</Text>
```

#### **3. Multi-line Text with Truncation**

```jsx
<Text numberOfLines={2} ellipsizeMode="tail">
  This is a very long text that will be truncated after two lines with three dots at the end...
</Text>
```

#### **4. Nested Text (Very Useful)**

```jsx
<Text style={styles.paragraph}>
  Hello, my name is <Text style={styles.bold}>Rajesh Sharma</Text> and I love{' '}
  <Text style={styles.highlight}>React Native</Text>!
</Text>
```

## 3. Image

**Description**: Displays images from local assets or remote URLs.

**Important**: For remote images, you **must** specify `width` and `height` (or use `resizeMode`).

### **What is Image?**
The `Image` component is used to display images in your React Native app. It supports:
- Local images (from `assets` folder)
- Remote images (from URLs)
- Different resize modes
- Placeholders and loading states

---

### **Basic Complete Example**

```jsx
import React from 'react';
import { Image, View, StyleSheet, Text } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      {/* Local Image */}
      <Text style={styles.label}>Local Image:</Text>
      <Image 
        source={require('./assets/logo.png')} 
        style={styles.localImage} 
      />

      {/* Remote Image */}
      <Text style={styles.label}>Remote Image:</Text>
      <Image 
        source={{ uri: 'https://picsum.photos/300/300' }}
        style={styles.remoteImage}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    alignItems: 'center',
    padding: 20,
  },
  label: {
    fontSize: 16,
    fontWeight: 'bold',
    marginTop: 20,
    marginBottom: 10,
  },
  localImage: {
    width: 150,
    height: 150,
    borderRadius: 10,
  },
  remoteImage: {
    width: 300,
    height: 300,
    borderRadius: 15,
  },
});

export default App;
```

---

### **Important Props**

| Prop              | What it does                                      | Common Values |
|-------------------|---------------------------------------------------|---------------|
| `source`          | Image source (local or remote)                    | `require()` or `{ uri: '...' }` |
| `style`           | Width, height, borderRadius, etc.                 | Style object |
| `resizeMode`      | How image fits in the container                   | `"cover"`, `"contain"`, `"stretch"`, `"center"` |
| `defaultSource`   | Placeholder image while loading (iOS)             | Local image |
| `loadingIndicatorSource` | Placeholder while loading (Android)        | Local image |

---

### **More Examples**

#### **1. Avatar Style Image**

```jsx
<Image
  source={{ uri: user.profilePic }}
  style={{
    width: 60,
    height: 60,
    borderRadius: 30,
    borderWidth: 2,
    borderColor: '#fff'
  }}
/>
```

#### **2. Image with Custom Resize Mode**

```jsx
<Image
  source={{ uri: 'https://picsum.photos/400' }}
  style={{ width: 300, height: 200 }}
  resizeMode="cover"        // or "contain"
/>
```

#### **3. Full Width Image with Aspect Ratio**

```jsx
<Image
  source={{ uri: 'https://picsum.photos/id/1015/800/400' }}
  style={{
    width: '100%',
    height: 220,
    borderRadius: 12
  }}
  resizeMode="cover"
/>
```

## 4. ScrollView

**Description**: Makes content scrollable. Best for small lists or static content.

### **What is ScrollView?**
`ScrollView` is a core React Native component that allows users to **scroll** through content that doesn’t fit on the screen. 

It’s ideal for:
- Static content (terms & conditions, about us, forms)
- Small number of items
- Horizontal scrolling (stories, image carousels)
- Content with mixed components

**Note**: For long/dynamic lists, use `FlatList` instead of `ScrollView` for better performance.

---

### **Basic Complete Example**

```jsx
import React from 'react';
import { ScrollView, Text, View, StyleSheet, Image } from 'react-native';

const App = () => {
  return (
    <ScrollView style={styles.container}>
      <View style={styles.content}>
        <Text style={styles.title}>Welcome to My App</Text>
        
        <Text style={styles.paragraph}>
          This is a long content that needs scrolling. 
          ScrollView makes it easy to view all the content.
        </Text>

        <Image 
          source={{ uri: 'https://picsum.photos/300/200' }} 
          style={styles.image} 
        />

        <Text style={styles.paragraph}>
          You can put Text, Images, Buttons, and many other components inside ScrollView.
        </Text>
      </View>
    </ScrollView>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
  },
  content: {
    padding: 20,
  },
  title: {
    fontSize: 28,
    fontWeight: 'bold',
    marginBottom: 15,
  },
  paragraph: {
    fontSize: 16,
    lineHeight: 24,
    marginBottom: 20,
  },
  image: {
    width: '100%',
    height: 200,
    borderRadius: 12,
    marginBottom: 20,
  },
});

export default App;
```

---

### **Important Props**

| Prop                | What it does                                      | Example |
|---------------------|---------------------------------------------------|--------|
| `horizontal`        | Enables horizontal scrolling                      | `true` |
| `refreshControl`    | Adds pull-to-refresh                              | `<RefreshControl ... />` |
| `showsVerticalScrollIndicator` | Shows/hides vertical scrollbar         | `false` |
| `showsHorizontalScrollIndicator` | Shows/hides horizontal scrollbar     | `false` |
| `contentContainerStyle` | Style for the inner content container       | Style object |

---

### **More Examples**

#### **1. Horizontal Scroll (Image Carousel)**

```jsx
<ScrollView horizontal showsHorizontalScrollIndicator={false}>
  <Image source={{ uri: 'https://picsum.photos/400/200' }} style={styles.image} />
  <Image source={{ uri: 'https://picsum.photos/401/200' }} style={styles.image} />
  <Image source={{ uri: 'https://picsum.photos/402/200' }} style={styles.image} />
</ScrollView>
```

#### **2. With Pull-to-Refresh**

```jsx
<ScrollView
  refreshControl={
    <RefreshControl refreshing={refreshing} onRefresh={onRefresh} />
  }
>
  {/* Your content */}
</ScrollView>
```

#### **3. Nested Scroll (Common Pattern)**

```jsx
<ScrollView>
  <Text style={styles.header}>Header Content</Text>
  
  <ScrollView horizontal nestedScrollEnabled={true}>
    {/* Horizontal items */}
  </ScrollView>

  <Text>More vertical content...</Text>
</ScrollView>
```

## 5. FlatList

**Description**: High-performance component for rendering long lists. Only renders visible items (virtualized).

### **What is FlatList?**
`FlatList` is the most commonly used component for displaying lists in React Native. It is highly optimized and only renders the items that are currently visible on the screen (virtualization), making it very performant even with hundreds or thousands of items.

Use `FlatList` for:
- User lists
- Product catalogs
- News feeds
- Chat messages
- Settings options

---

### **Basic Complete Example**

```jsx
import { FlatList, StyleSheet, Text, View } from 'react-native';
import React from 'react';

const users = [
  { id: '1', name: 'Rajesh' },
  { id: '2', name: 'Priya' },
  { id: '3', name: 'Aman' },
  { id: '4', name: 'Sneha' },
  { id: '5', name: 'Rohit' },
  { id: '6', name: 'Vikash' },
  { id: '7', name: 'Sumit' },
];

const App = () => {
  return (
    <View style={styles.container}>
      <Text style={styles.heading}>Users</Text>

      <FlatList
        data={users}
        keyExtractor={(item) => item.id}
        showsVerticalScrollIndicator={false}
        renderItem={({ item }) => (
          <View style={styles.card}>
            <View style={styles.avatar}>
              <Text style={styles.avatarText}>
                {item.name.charAt(0)}
              </Text>
            </View>

            <Text style={styles.name}>{item.name}</Text>
          </View>
        )}
      />
    </View>
  );
};

export default App;

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#F5F7FA',
    paddingHorizontal: 20,
    paddingTop: 50,
  },

  heading: {
    fontSize: 28,
    fontWeight: '700',
    color: '#222',
    marginBottom: 20,
  },

  card: {
    backgroundColor: '#fff',
    flexDirection: 'row',
    alignItems: 'center',
    padding: 16,
    borderRadius: 14,
    marginBottom: 12,
    shadowColor: '#000',
    shadowOpacity: 0.08,
    shadowRadius: 8,
    shadowOffset: {
      width: 0,
      height: 3,
    },
    elevation: 3,
  },

  avatar: {
    width: 50,
    height: 50,
    borderRadius: 25,
    backgroundColor: '#4F46E5',
    justifyContent: 'center',
    alignItems: 'center',
    marginRight: 15,
  },

  avatarText: {
    color: '#fff',
    fontSize: 20,
    fontWeight: 'bold',
  },

  name: {
    fontSize: 18,
    color: '#333',
    fontWeight: '600',
  },
});
```

---

### **Important Props**

| Prop                    | What it does                                      | Example |
|-------------------------|---------------------------------------------------|--------|
| `data`                  | Array of data to display                          | `users` |
| `renderItem`            | Function to render each item                      | `({ item }) => <Text>{item.name}</Text>` |
| `keyExtractor`          | Returns unique key for each item                  | `(item) => item.id` |
| `ItemSeparatorComponent`| Component rendered between items                  | `() => <View style={{height: 1}} />` |
| `ListHeaderComponent`   | Component at the top of the list                  | `<Text>Header</Text>` |
| `ListFooterComponent`   | Component at the bottom of the list               | `<Text>End of list</Text>` |
| `refreshControl`        | Pull-to-refresh functionality                     | `<RefreshControl ... />` |

---

### **More Examples**


```jsx
<FlatList
```

```jsx
  horizontal={true}
  renderItem={({ item }) => <HorizontalCard user={item} />}

/>
```

## 6. SectionList

**Description**: Used for grouped (sectioned) lists like contacts or settings.

### **What is SectionList?**
`SectionList` is a performant React Native component used to display **grouped/organized lists**. It’s perfect for:
- Contacts list (A, B, C sections)
- Settings page with categories
- Country or city lists
- Product categories

It has built-in support for **section headers** and is more efficient than using multiple FlatLists.

---

### **Basic Complete Example**

```jsx
import React from 'react';
import { SectionList, Text, View, StyleSheet } from 'react-native';

const sections = [
  {
    title: 'A',
    data: ['Aman', 'Ankit', 'Arjun'],
  },
  {
    title: 'B',
    data: ['Bharat', 'Bobby', 'Binod'],
  },
  {
    title: 'C',
    data: ['Chirag', 'Chetan'],
  },
];

const App = () => {
  return (
    <View style={styles.container}>
      <SectionList
        sections={sections}
        keyExtractor={(item, index) => item + index}
        renderItem={({ item }) => (
          <View style={styles.item}>
            <Text style={styles.itemText}>{item}</Text>
          </View>
        )}
        renderSectionHeader={({ section }) => (
          <View style={styles.header}>
            <Text style={styles.headerText}>{section.title}</Text>
          </View>
        )}
        stickySectionHeadersEnabled={true}   // Header stays on top while scrolling
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
  },
  header: {
    backgroundColor: '#f0f0f0',
    padding: 15,
    borderBottomWidth: 1,
    borderBottomColor: '#ddd',
  },
  headerText: {
    fontSize: 18,
    fontWeight: 'bold',
  },
  item: {
    padding: 15,
    borderBottomWidth: 1,
    borderBottomColor: '#eee',
  },
  itemText: {
    fontSize: 16,
  },
});

export default App;
```

---

### **Important Props**

| Prop                        | What it does                                      | Example Value |
|----------------------------|---------------------------------------------------|---------------|
| `sections`                 | Array of sections with title and data             | `[{title, data}]` |
| `renderItem`               | How each item in the list looks                   | `({item}) => <Text>{item}</Text>` |
| `renderSectionHeader`      | How each section header looks                     | `({section}) => <Text>{section.title}</Text>` |
| `keyExtractor`             | Unique key for each item                          | `(item, index) => item + index` |
| `stickySectionHeadersEnabled` | Keeps header visible while scrolling         | `true` / `false` |
| `onRefresh` + `refreshing` | Pull to refresh support                           | Same as FlatList |

---

### **More Examples**

#### **1. With Section Separator**

```jsx
<SectionList
  sections={sections}
  renderItem={...}
  renderSectionHeader={...}
  SectionSeparatorComponent={() => <View style={{ height: 1, backgroundColor: '#ccc' }} />}
/>
```

#### **2. With Header and Footer**

```jsx
<SectionList
  sections={sections}
  renderItem={...}
  renderSectionHeader={...}
  ListHeaderComponent={<Text style={styles.mainHeader}>My Contacts</Text>}
  ListFooterComponent={<Text style={styles.footer}>End of List</Text>}
/>
```

## 7. Pressable

**Description**: Modern and highly customizable component for handling touch events.

### **What is Pressable?**
`Pressable` is a newer and more powerful alternative to `TouchableOpacity`. It gives you **better control** over touch interactions and allows you to change styles dynamically based on whether the button is being pressed.

It is recommended for modern React Native apps because it’s more flexible and performant.

---

### **Basic Complete Example**

```jsx
import React from 'react';
import { Pressable, Text, View, StyleSheet, Alert } from 'react-native';

const App = () => {
  const handlePress = () => {
    Alert.alert('Success', 'Button was pressed! ');
  };

  return (
    <View style={styles.container}>
      <Pressable 
        style={styles.button}
        onPress={handlePress}
      >
        <Text style={styles.buttonText}>Press Me</Text>
      </Pressable>
    </View>
  );
};

const styles = StyleSheet.create({
```

## 14. RefreshControl

**Description**: Adds pull-to-refresh functionality.

### **What is RefreshControl?**
`RefreshControl` is a built-in React Native component that provides **pull-to-refresh** functionality. When the user pulls down on the screen, a loading spinner appears and the data gets refreshed — just like you see in Instagram, Twitter/X, WhatsApp, etc.

It is commonly used with scrollable components like **ScrollView**, **FlatList**, and **SectionList**.

---

### **Complete Basic Example (with FlatList)**

```jsx
import React, { useState } from 'react';
import { FlatList, RefreshControl, Text, View, StyleSheet } from 'react-native';

const App = () => {
  const [refreshing, setRefreshing] = useState(false);
  const [data, setData] = useState(['Item 1', 'Item 2', 'Item 3', 'Item 4']);

  const onRefresh = async () => {
    setRefreshing(true);        // Start the loader

    // Put your API call or data refresh logic here
    await new Promise(resolve => setTimeout(resolve, 2000)); // 2 second fake delay

    // Update data (in real apps, this comes from API)
    setData(['New Item 1', ...data]);

    setRefreshing(false);       // Stop the loader
  };

  return (
    <FlatList
      data={data}
      keyExtractor={(item, index) => index.toString()}
      renderItem={({ item }) => (
        <View style={styles.item}>
          <Text style={styles.text}>{item}</Text>
        </View>
      )}
      refreshControl={
        <RefreshControl
          refreshing={refreshing}
          onRefresh={onRefresh}
          colors={['#ff0000', '#00ff00', '#0000ff']} // Android
          tintColor="#ff0000"                        // iOS
          title="Refreshing..."                      // iOS
        />
      }
    />
  );
};

const styles = StyleSheet.create({
  item: {
    padding: 20,
    borderBottomWidth: 1,
    borderBottomColor: '#ddd',
  },
  text: {
    fontSize: 18,
  },
});

export default App;
```

---

### **Important Props**

| Prop                        | What it does                                      | Example Value                     |
|----------------------------|---------------------------------------------------|-----------------------------------|
| `refreshing`               | Controls whether the loader is visible            | `true` / `false`                  |
| `onRefresh`                | Function called when user pulls to refresh        | `() => {}`                        |
| `colors`                   | Array of colors for the loader (Android)          | `['red', 'blue', 'green']`        |
| `tintColor`                | Color of the loader indicator (iOS)               | `'#ff0000'`                       |
| `title`                    | Text shown below the loader (iOS)                 | `'Updating...'`                   |
| `titleColor`               | Color of the title text (iOS)                     | `'#000'`                          |
| `progressBackgroundColor`  | Background color behind the loader (Android)      | `'#ffffff'`                       |
| `size`                     | Size of the loader (Android only)                 | `RefreshControl.SIZE.LARGE`       |

---

### **More Examples**

#### **1. Using with ScrollView**

```jsx

  refreshControl={
    <RefreshControl refreshing={refreshing} onRefresh={onRefresh} />
  }
>
  {/* Your content goes here */}
```

## 15. TextInput

**Description**: The primary component for collecting user text input in React Native. Supports single-line, multi-line, passwords, and various keyboard types.

### What is TextInput?
`TextInput` is React Native’s equivalent of the HTML `<input>` and `<textarea>` elements. It is **essential** for forms, login screens, search bars, chat inputs, and almost any user data collection.

---

### Basic Complete Example (Single Line)

```jsx
import React, { useState } from 'react';
import { TextInput, View, Text, StyleSheet } from 'react-native';

const App = () => {
  const [name, setName] = useState('');

  return (
    <View style={styles.container}>
      <Text style={styles.label}>Full Name</Text>
      <TextInput
        style={styles.input}
        placeholder="Enter your full name"
        value={name}
        onChangeText={setName}
      />
      <Text style={styles.output}>Entered: {name || 'Nothing yet'}</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: { flex: 1, padding: 20, justifyContent: 'center' },
  label: { fontSize: 18, marginBottom: 8, fontWeight: '600' },
  input: {
    height: 55,
    borderWidth: 1,
    borderColor: '#ddd',
    borderRadius: 10,
    paddingHorizontal: 15,
    fontSize: 16,
    backgroundColor: '#fff',
  },
  output: { marginTop: 15, fontSize: 16 },
});

export default App;
```

---

### Important Props (Most Used)

| Prop                    | What it does                                           | Common Values |
|-------------------------|--------------------------------------------------------|---------------|
| `value`                 | Current text (controlled)                              | string |
| `onChangeText`          | Called every time text changes                         | `(text) => {}` |
| `placeholder`           | Text shown when input is empty                         | string |
| `secureTextEntry`       | Masks input (passwords)                                | `true` / `false` |
| `keyboardType`          | Controls keyboard layout                               | `"default"`, `"numeric"`, `"email-address"`, `"phone-pad"` |
| `multiline`             | Allows multiple lines of text                          | `true` / `false` |
| `numberOfLines`         | Number of lines to show (with multiline)               | number |
| `autoCapitalize`        | Controls automatic capitalization                      | `"none"`, `"sentences"`, `"words"`, `"characters"` |
| `autoCorrect`           | Enables/disables auto-correction                       | `true` / `false` |
| `placeholderTextColor`  | Color of placeholder text                              | color string |
| `maxLength`             | Maximum characters allowed                             | number |
| `editable`              | Whether user can edit the text                         | `true` / `false` |
