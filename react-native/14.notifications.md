# 1️⃣ What is Networking?

Networking is the process of exchanging data between your app and a server.

Common examples:

- Login
- Upload Profile Picture
- Download PDF
- Fetch Products
- Chat Messages

# 2️⃣ File Upload

## Common Use Cases

- Profile Photo
- Resume Upload
- PDF Upload
- Multiple Images
- KYC Documents

### Flow

```text
Pick File
    │
Create FormData
    │
POST Request
    │
Backend
    │
Upload Success
```

Example (Axios):

```jsx
const formData = new FormData();

formData.append("image", {
  uri: image.uri,
  name: "photo.jpg",
  type: "image/jpeg",
});

await axios.post("/upload", formData, {
  headers: {
    "Content-Type": "multipart/form-data",
  },
});
```

# 3️⃣ FormData

`FormData` is used to send files and form fields together.

```jsx
const formData = new FormData();

formData.append("name", "Rajesh");
formData.append("email", "demo@gmail.com");
```

### Supported Types

- Images
- Videos
- PDFs
- Audio
- Text Fields

# 4️⃣ File Download

Download files from a server.

Expo:

```bash
npx expo install expo-file-system
```

Example:

```jsx
import * as FileSystem from "expo-file-system";

const result =
await FileSystem.downloadAsync(
  fileUrl,
  FileSystem.documentDirectory + "sample.pdf"
);

console.log(result.uri);
```

### Download Flow

```text
User Clicks Download
        │
HTTP Request
        │
Server Response
        │
Save File
        │
Open File
```

# 5️⃣ NetInfo

Detect internet connectivity.

Install:

```bash
npx expo install @react-native-community/netinfo
```

Example:

```jsx
import NetInfo from "@react-native-community/netinfo";

const state = await NetInfo.fetch();

console.log(state.isConnected);
```

Listen for changes:

```jsx
const unsubscribe = NetInfo.addEventListener(state => {
  console.log(state.isConnected);
});

// Cleanup
unsubscribe();
```

# 6️⃣ Offline Mode

Apps should continue working even without internet.

Strategies:

- Cache API responses
- Queue pending requests
- Retry when online
- Show offline message

Flow:

```text
Internet Available?
      │
  Yes ─────► API Request
      │
      No
      │
Read Cached Data
      │
Retry Later
```

---

# 7️⃣ Retry Failed Requests

```jsx
if (!isConnected) {
  alert("No Internet Connection");
  return;
}
```

Retry automatically when connectivity returns.

# 8️⃣ Best Practices

- Compress files before upload
- Show upload/download progress
- Validate file size
- Handle timeouts
- Detect offline state
- Cancel duplicate requests
- Store pending data locally
