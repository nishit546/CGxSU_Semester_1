# ğŸš€ Expo Router (File-Based Navigation)

Expo Router is a **modern navigation system** for Expo apps that uses **file and folder-based routing**, similar to **Next.js**.

Instead of manually configuring navigators (Stack, Tab, Drawer),  
**your folder structure becomes your navigation**.

---

## What is Expo Router?

**Expo Router** is an official Expo library built **on top of React Navigation**.

It allows you to:
- Create screens using files
- Navigate automatically based on file names
- Reduce boilerplate navigation code
- Build scalable apps easily

> â„¹ï¸ Even though Expo Router hides configuration, it internally uses **React Navigation**.

---

## Why Expo Router Exists

Traditional navigation:
- Requires manual Stack/Tab/Drawer setup
- Needs multiple config files
- Can be confusing for beginners

Expo Router:
- Uses files as screens
- Automatic navigation setup
- Cleaner and more intuitive for new projects

---

## ğŸ”€ Dynamic Routes (Passing Params)
