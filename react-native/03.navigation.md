## What is React Navigation?

**React Navigation** is the most popular navigation library for React Native.

It helps you:
- Navigate between screens
- Pass data between screens
- Handle back navigation
- Create stack, tab, and drawer layouts

## ğŸ“š Stack Navigation (Most Common)

Stack navigation works like a **stack of screens**:

- New screen is pushed on top
- Back button pops the screen

### Example Use Case

Login â†’ Home â†’ Details

### Installing Stack Navigator

```bash
npm install @react-navigation/stack
```

## ğŸ”° Basic Stack Navigation Example

```jsx
import { createStackNavigator } from "@react-navigation/stack";
import { NavigationContainer } from "@react-navigation/native";
import { View, Text, Pressable } from "react-native";

const Stack = createStackNavigator();

function HomeScreen({ navigation }) {
  return (
    <View>
      <Text>Home Screen</Text>
      <Pressable onPress={() => navigation.navigate("Details")}>
        <Text>Go to Details</Text>
      </Pressable>
    </View>
  );
}

function DetailsScreen() {
  return (
    <View>
      <Text>Details Screen</Text>
    </View>
  );
}

export default function App() {
  return (
    <NavigationContainer>
      <Stack.Navigator>
        <Stack.Screen name="Home" component={HomeScreen} />
        <Stack.Screen name="Details" component={DetailsScreen} />
      </Stack.Navigator>
    </NavigationContainer>
  );
}
```

## ğŸ§­ navigation Prop (Very Important)

Each screen automatically receives a **navigation** prop.

### Common methods:

- `navigation.navigate("ScreenName")`
- `navigation.goBack()`
- `navigation.replace("ScreenName")`

### Example:

```jsx
navigation.navigate("Profile");
```
---
## ğŸ“¦ route Prop (Accessing Data)

The **route** prop is used to access **parameters** passed between screens.

### Passing Data

```jsx
navigation.navigate("Details", { id: 10 });
```
---
### Receiving Data

```jsx
function DetailsScreen({ route }) {
  const { id } = route.params;

  return <Text>Item ID: {id}</Text>;
}
```

## â“ Which Navigation Should You Use?

- Use **Tab Navigation** for main sections of the app
- Use **Drawer Navigation** for global menus and settings
