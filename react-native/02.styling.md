# üé® Styling in React Native

Styling in React Native is used to control **layout, colors, spacing, fonts, and responsiveness** of your app UI.

Unlike web development, React Native **does not use CSS files**.  
Instead, styles are written using **JavaScript objects**.

---

## How Styling Works in React Native

In React Native:
- Styles are written in **JavaScript**
- Styling is applied using the `style` prop
- Uses **Flexbox by default**
- Styling works consistently across platforms
  
Example:
```jsx
<Text style={{ color: "blue", fontSize: 20 }}>
  Hello World
</Text>
```

## üß© StyleSheet API (Recommended Way)

React Native provides a built-in **StyleSheet API** to define styles.

### Why Use StyleSheet?

- Cleaner code
- Better readability
- Slight performance benefits
- Easy reuse of styles

### Example

```jsx
import { View, Text, StyleSheet } from "react-native";

function App() {
  return (
    <View style={styles.container}>
      <Text style={styles.text}>Styled Text</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    padding: 20,
    backgroundColor: "#f2f2f2",
  },
  text: {
    fontSize: 18,
    color: "blue",
  },
});

export default App;
```

## üìê Flexbox in React Native (Very Important)

React Native uses **Flexbox by default** for layout.

### Default Values

- `flexDirection`: column
- `justifyContent`: flex-start
- `alignItems`: stretch

```jsx
<View style={{ flexDirection: "row" }}>
  <Text>One</Text>
  <Text>Two</Text>
</View>
```
---

### flex

- `flex: 1` makes a component take available space
- Higher values take more space proportionally

```jsx
<View style={{ flex: 1 }}>
  <Text>Full Screen</Text>
</View>
```
---

### justifyContent (Main Axis)

- flex-start
- center
- space-between
- space-around

 ```jsx
<View style={{ flexDirection: "row", justifyContent: "space-between" }}>
  <Text>Left</Text>
  <Text>Right</Text>
</View>
```
--- 
### alignItems (Cross Axis)
 ```jsx
<View style={{ alignItems: "center" }}>
  <Text>Centered</Text>
</View>
```

## üì± Responsive Styling

React Native supports responsive UI using:

- Dimensions
- flex
- Percentage values

### Example using Dimensions

```js
import { Dimensions, StyleSheet } from "react-native";

const screenWidth = Dimensions.get("window").width;

const styles = StyleSheet.create({
  box: {
    width: screenWidth * 0.8,
    height: 100,
    backgroundColor: "orange",
  },
});
```

## üß© Combining Multiple Styles

You can pass **multiple styles** using an array.

```jsx
<Text style={[styles.text, styles.bold]}>
  Combined Styles
</Text>
```
```jsx
const styles = StyleSheet.create({
  text: {
    fontSize: 16,
  },
  bold: {
    fontWeight: "bold",
  },
});
```
> ‚ÑπÔ∏è When multiple styles are combined, the **last style in the array overrides previous ones**.
---

## üß≠ Platform-Specific Styling (Intro)

You can apply styles differently for **Android** and **iOS**.

```js
import { Platform, StyleSheet } from "react-native";

const styles = StyleSheet.create({
  text: {
    fontSize: Platform.OS === "ios" ? 18 : 16,
  },
});
```

## ‚ùå Common Styling Mistakes (Avoid These)

- ‚ùå Using CSS files
- ‚ùå Forgetting Flexbox defaults
- ‚ùå Overusing inline styles
- ‚ùå Not giving width/height to Image

---

## ‚úÖ Best Practices for Styling

- Use `StyleSheet.create`
- Keep styles in one place
- Use meaningful style names
- Prefer Flexbox for layout
- Keep UI responsive

- StyleSheet improves readability and reuse
