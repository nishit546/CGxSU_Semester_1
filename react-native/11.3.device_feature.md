# 1️⃣ Clipboard (`expo-clipboard`)

## What is Clipboard?

Clipboard is temporary storage used for copying and pasting data.

### Common Use Cases

- Copy OTP
- Copy Coupon Code
- Copy Referral Code
- Copy URL
- Copy Email Address

---

## Installation

```bash
npx expo install expo-clipboard
```

---

## Copy Text

```jsx
import * as Clipboard from "expo-clipboard";

await Clipboard.setStringAsync("WELCOME50");
```

---

## Read Clipboard

```jsx
const value = await Clipboard.getStringAsync();

console.log(value);
```

---

## Practical Example

```jsx
<Pressable
  onPress={() =>
    Clipboard.setStringAsync("https://myapp.com")
  }
>
  <Text>Copy Link</Text>
</Pressable>
```

# 2️⃣ Share API

## What is Share API?

Share API allows users to share content with other applications.

Examples:

- WhatsApp
- Gmail
- Telegram
- Instagram
- Messages

---

## Share Text

```jsx
import { Share } from "react-native";

await Share.share({
  message: "Check out my React Native App!",
});
```

---

## Share URL

```jsx
await Share.share({
  message: "https://myapp.com",
});
```

---

## Share Image or File (Expo)

Install:

```bash
npx expo install expo-sharing
```

Example:

```jsx
import * as Sharing from "expo-sharing";

await Sharing.shareAsync(fileUri);
```

---

## Share Flow

```text
User Clicks Share
        │
Prepare Content
        │
Open Native Share Sheet
        │
User Selects App
        │
Content Shared
```

# 3️⃣ Haptics (`expo-haptics`)

## What are Haptics?

Haptics provide physical vibration feedback for user interactions.

### Real-world Examples

- Button Press
- Payment Success
- Login Success
- Error Alert
- Form Validation

---

## Installation

```bash
npx expo install expo-haptics
```

---

## Selection Feedback

```jsx
import * as Haptics from "expo-haptics";

await Haptics.selectionAsync();
```

---

## Success Notification

```jsx
await Haptics.notificationAsync(
  Haptics.NotificationFeedbackType.Success
);
```

---

## Error Notification

```jsx
await Haptics.notificationAsync(
  Haptics.NotificationFeedbackType.Error
);
```

---

## Warning Notification

```jsx
await Haptics.notificationAsync(
  Haptics.NotificationFeedbackType.Warning
);
```

---

## Impact Feedback

```jsx
await Haptics.impactAsync(
  Haptics.ImpactFeedbackStyle.Heavy
);
```

Styles:

- Light
- Medium
- Heavy
- Rigid
- Soft

---

#  When to Use Haptics

| Event | Feedback |
|--------|----------|
| Button Click | Selection |
| Login Success | Success |
| Payment Failed | Error |
| Delete Warning | Warning |
| Drag & Drop | Light Impact |

#  Production Best Practices

## Clipboard

- Copy only useful data
- Show a confirmation message
- Avoid copying sensitive information

## Share API

- Validate content before sharing
- Handle unsupported file types
- Check file existence

## Haptics

- Don't vibrate on every action
- Use subtle feedback
- Respect platform behavior

## General

- Handle permissions gracefully
- Test on Android and iOS
- Avoid blocking the UI
- Handle offline scenarios
- Show loading indicators

---

#  Common Mistakes

- Forgetting async/await
- Sharing invalid files
- Excessive haptic feedback
- Ignoring user cancellations
- No error handling
