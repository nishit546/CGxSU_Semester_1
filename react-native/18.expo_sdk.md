# 1️⃣ What is the Expo SDK?

The Expo SDK is a collection of officially maintained libraries that provide access to native device features without writing Android (Java/Kotlin) or iOS (Swift/Objective-C) code.

### Benefits

- Easy installation
- Cross-platform support
- Well documented
- Works seamlessly with Expo

Install packages using:

```bash
npx expo install <package-name>
```

# 2️⃣ expo-camera

Access the device camera.

## Install

```bash
npx expo install expo-camera
```

### Features

- Camera Preview
- Capture Photos
- Front / Back Camera
- Flash
- Zoom
- Barcode Scanning

### Common Use Cases

- QR Scanner
- Profile Picture
- Document Scanner

# 8️⃣ expo-router

File-based routing for Expo applications.

## Install

```bash
npx expo install expo-router
```

### Folder Example

```text
app/
├── index.tsx
├── profile.tsx
├── settings.tsx
└── product/
    └── [id].tsx
```

### Benefits

- File-based routing
- Deep linking support
- Layouts
- Nested routes

#  Best Practices

- Install packages with `expo install`
- Request permissions only when required
- Keep Expo SDK versions aligned
- Read release notes before upgrading
- Test features on real devices

---

#  Common Mistakes

- Using `npm install` instead of `expo install`
- Ignoring permission handling
- Mixing incompatible SDK versions
- Forgetting OTA update testing
