# Chapter 1 – Camera (`expo-camera`)

## What is expo-camera?

`expo-camera` is the official Expo library for accessing the device camera.

### Real-world Use Cases

- QR Scanner
- Attendance App
- Document Scanner
- Social Media Apps
- Profile Picture Capture

---

## Installation

```bash
npx expo install expo-camera
```

---

## Camera Permissions

```jsx
import { CameraView, useCameraPermissions } from "expo-camera";

const [permission, requestPermission] = useCameraPermissions();

if (!permission?.granted) {
  return <Button title="Grant Permission" onPress={requestPermission} />;
}
```

### Permission Flow

```text
App Opens
    │
Request Camera Permission
    │
Granted? ─── No ───► Show Permission Screen
    │
   Yes
    │
Open Camera
```

---

## CameraView

```jsx
<CameraView
  style={{ flex: 1 }}
  facing="back"
/>
```

### Camera Types

- back
- front

---

## Capturing an Image

```jsx
const photo = await cameraRef.current.takePictureAsync();

console.log(photo.uri);
```

Returned data:

- uri
- width
- height

---

## Flash Modes

- off
- on
- auto

---

## Zoom

```jsx
<CameraView zoom={0.5} />
```

---

## Best Practices

- Ask permission only when required.
- Stop camera when leaving the screen.
- Compress large images.
- Handle denied permissions gracefully.

# Chapter 2 – Gallery (`expo-image-picker`)

## Installation

```bash
npx expo install expo-image-picker
```

---

## Gallery Permission

```jsx
const permission =
await ImagePicker.requestMediaLibraryPermissionsAsync();
```

---

## Pick Single Image

```jsx
const result =
await ImagePicker.launchImageLibraryAsync({
  mediaTypes: ["images"],
  quality: 0.8,
});
```

---

## Pick Multiple Images

```jsx
const result =
await ImagePicker.launchImageLibraryAsync({
  allowsMultipleSelection: true,
});
```

---

## Crop Images

```jsx
allowsEditing: true,
aspect: [1,1]
```

---

## Preview Image

```jsx
<Image
 source={{ uri: result.assets[0].uri }}
 style={{ width: 200, height: 200 }}
/>
```

---

## Compress Images

```jsx
quality: 0.5
```

---

## Handle Cancel

```jsx
if (result.canceled) {
  return;
}
```

---

## Image Metadata

Every selected image provides:

- uri
- width
- height
- fileName
- mimeType

---

## Prepare Image Upload

```jsx
const formData = new FormData();

formData.append("image", {
  uri: image.uri,
  type: "image/jpeg",
  name: "photo.jpg",
});
```

---

# Camera vs Gallery

| Camera | Gallery |
|---------|----------|
| Capture new photo | Select existing image |
| Requires Camera Permission | Requires Gallery Permission |
| Real-time | Previously stored |

---

# Best Practices

- Compress images before upload.
- Show loading indicators.
- Validate file size.
- Preview before upload.
- Cache images where appropriate.

---

# Common Mistakes

- Forgetting permissions
- Uploading original 10MB+ images
- Ignoring cancel action
- Missing error handling
