# 1️⃣ What is an API?

API (Application Programming Interface) allows your app to communicate with a server.

Example Flow:

```text
React Native App
        │
HTTP Request
        │
Backend API
        │
Database
        │
JSON Response
        │
Mobile App
```

# 3️⃣ Using fetch()

### GET Request

```jsx
useEffect(() => {
  fetch("https://jsonplaceholder.typicode.com/users")
    .then((res) => res.json())
    .then((data) => console.log(data))
    .catch((err) => console.log(err));
}, []);
```

### POST Request

```jsx
fetch("https://jsonplaceholder.typicode.com/posts", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    title: "React Native",
    body: "Learning API",
  }),
});
```

# 4️⃣ Axios

Install:

```bash
npm install axios
```

### GET

```jsx
import axios from "axios";

const res = await axios.get(
  "https://jsonplaceholder.typicode.com/users"
);

console.log(res.data);
```

### POST

```jsx
await axios.post("/users", {
  name: "Rajesh",
  email: "demo@gmail.com",
});
```

### Why Axios?

- Cleaner syntax
- Automatic JSON parsing
- Better error handling
- Request/Response interceptors
- Timeout support

---

# 5️⃣ PUT Request

Replace an entire resource.

```jsx
await axios.put("/users/1", {
  name: "Updated User",
});
```

---

# 6️⃣ PATCH Request

Update selected fields.

```jsx
await axios.patch("/users/1", {
  email: "new@email.com",
});
```

---

# 7️⃣ DELETE Request

```jsx
await axios.delete("/users/1");
```

# 9️⃣ Error Handling

```jsx
try {
  const res = await axios.get("/users");
} catch (error) {
  console.log(error.message);
}
```

Common Errors

- Network Error
- 401 Unauthorized
- 404 Not Found
- 500 Server Error

# 🔟 Authorization Headers

JWT Token Example:

```jsx
const token = "your_token";

await axios.get("/profile", {
  headers: {
    Authorization: `Bearer ${token}`,
  },
});
```

Store tokens securely using:

- AsyncStorage (basic)
- SecureStore (recommended)

# 1️⃣1️⃣ Axios Instance

```jsx
import axios from "axios";

export const api = axios.create({
  baseURL: "https://api.example.com",
  timeout: 10000,
});
```

Usage:

```jsx
const res = await api.get("/users");
```

---

# 1️⃣2️⃣ Axios Interceptors

Automatically attach tokens.

```jsx
api.interceptors.request.use((config) => {
  config.headers.Authorization = "Bearer TOKEN";
  return config;
});
```

Useful for:

- Authentication
- Logging
- Refresh Tokens

#  fetch vs Axios

| Feature | fetch | Axios |
|---------|-------|-------|
| Built-in | ✅ | ❌ |
| Auto JSON | ❌ | ✅ |
| Interceptors | ❌ | ✅ |
| Timeout | ❌ | ✅ |
| Cleaner Syntax | ❌ | ✅ |

#  Best Practices

- Create a single Axios instance
- Keep API URLs in one place
- Use async/await
- Handle loading and errors
- Never hardcode tokens
- Validate API responses

---

#  Common Mistakes

- Forgetting `await`
- Not handling errors
- Showing no loading state
- Hardcoding URLs
- Exposing tokens
- Making duplicate API requests
