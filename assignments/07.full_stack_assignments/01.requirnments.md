## **5. Advanced Coding Video Assignment (Most Important)**

Students must create:

* **30 coding videos (5–10 minutes each)**

### **Focus Areas**

* React
* Node.js
* MongoDB

### **Category Split**

* React: **12 videos**
* Node.js: **12 videos**
* MongoDB: **6 videos**

---

### **Video Rules**

* Language: **English only**

* Face must be clearly visible

* Start from **blank code (no copy-paste)**

* Explain:

  * Problem
  * Approach
  * Code while writing

* Show output at the end

---

### **Content Rules**

* Questions will be from a **provided list (60–80 questions)**
* Questions will be:

  * Commonly asked in interviews
  * Solvable within **5–10 minutes**

---

### **Title & Description**

* Title = exact question name
* Description must include:

  * Approach
  * GitHub
  * Portfolio
  * LinkedIn

---

* 3 videos per week (approx.)
