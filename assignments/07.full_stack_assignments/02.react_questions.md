# **React Interview Questions (Fresher | Practical | 5–10 min)**

## **State & UI Control (Core Basics)**

1. **Counter with Constraints**
   Create a counter with:

   * Increment (+1)
   * Decrement (-1, but not below 0)
   * Reset button

2. **Toggle Visibility**
   Create a button that shows/hides a paragraph when clicked

3. **Disable Button After Click**
   Create a button that becomes disabled after first click

4. **Like Button**
   Create a "Like" button:

   * Toggle between "Like" and "Liked"
   * Show total likes count

---

## **Forms (Very Common in Interviews)**

5. **Controlled Input (Single Field)**
   Create an input field and display its value live below

6. **Form with Multiple Inputs**
   Create a form with:

   * Name
   * Email
   * On submit → log data

7. **Basic Form Validation**
   Same form:

   * Name required
   * Email must contain `@`
   * Show error messages

8. **Disable Submit Until Valid**
   Disable submit button until all fields are valid

---

## **Lists & Rendering**

9. **Render Static List**
   Display a list of items using `map()`

10. **Dynamic List Add Item**
    Add items to a list using input + button

11. **Delete Item from List**
    Add delete button for each item

12. **Mark Item as Completed**
    Toggle completed (strike-through text)

---

## **Component Communication**

13. **Parent → Child Data Passing**
    Pass name from parent to child and display

14. **Child → Parent Communication**
    Child button sends data back to parent

15. **Reusable Button Component**
    Create a button component with props:

* label
* onClick

## **Conditional Rendering**

16. **Login / Logout UI**
    Toggle between:

17. **Show Loader While Fetching**
    Show "Loading..." before data appears

18. **Fetch and Display Data**
