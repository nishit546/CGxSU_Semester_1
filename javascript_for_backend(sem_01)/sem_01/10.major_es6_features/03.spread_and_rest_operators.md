# **Spread & Rest Operators ( … ) in JavaScript**

# **1. Introduction to the “…” Operator**

JavaScript uses the same symbol `...` for two different but related features:

* **Spread Operator** → expands/unpacks values
* **Rest Operator** → collects/groups values

They look the same but their purpose depends on **where** they are used.

```
... in function call → Spread  
... in array literal → Spread  
... in object literal → Spread  

... in function parameter → Rest  
... in array destructuring → Rest  
... in object destructuring → Rest
```

Understanding this properly is essential for professional JavaScript.

---

# **2. Spread Operator ( ... ) — “Expand / Unpack”**

Spread operator **expands** an iterable (array, string, object) into individual elements.

---

# **2.1 Spread with Arrays**

### Example 1 — Copying an array (shallow copy)

```javascript
var a = [1, 2, 3];
var b = [...a];

console.log(b);       // [1, 2, 3]
console.log(a === b); // false (different arrays)
```

---

### Example 2 — Merging arrays

```javascript
var html = ["HTML", "Tags"];
var css = ["CSS", "Selectors"];

var topics = [...html, ...css];

console.log(topics);
```

Output:

```
["HTML", "Tags", "CSS", "Selectors"]
```

---

### Example 3 — Adding elements in between

```javascript
var arr = [1, 2, 3];

var result = [0, ...arr, 4];

console.log(result); // [0, 1, 2, 3, 4]
```

---

### Example 4 — Convert string → array

```javascript
var name = "Krishna";

var letters = [...name];

console.log(letters);
// ["K", "r", "i", "s", "h", "n", "a"]
```

---

# **2.2 Spread with Objects**

Spread also works for **objects** (ES2018):

### Example 1 — Copy object

```javascript
var student = { name: "Arjun", age: 18 };

var newStudent = { ...student };

console.log(newStudent);
```

---

### Example 2 — Merge objects

```javascript
var info = { name: "Mahir" };
var marks = { html: 90, css: 88 };

var student = { ...info, ...marks };

console.log(student);
```

---

### Example 3 — Add new properties

```javascript
var student = { name: "Krishna" };

var updated = { ...student, branch: "CSE" };

console.log(updated);
```

---

### Example 4 — Overwrite properties

```javascript
var student = { name: "Jenil", age: 19 };

var updated = { ...student, age: 20 };

console.log(updated.age); // 20
```

---

# **2.3 Spread in Function Calls**

Spread expands arrays into separate parameters.

### Example

```javascript
function sum(a, b, c) {
    return a + b + c;
}

var nums = [10, 20, 30];

console.log(sum(...nums)); // 60
```

---

### Real Example — Maximum number

```javascript
var marks = [85, 92, 78];

console.log(Math.max(...marks)); // 92
```

---

# **3. Rest Operator ( ... ) — “Collect / Group”**

Rest operator collects multiple values into a **single array** or **object**.

---

# **3.1 Rest in Function Parameters**

It is used to accept unlimited arguments.

### Example:

```javascript
function total(...nums) {
    var sum = 0;
    for (var n of nums) sum += n;
    return sum;
}

console.log(total(10, 20, 30, 40)); // 100
```

Here `...nums` gathers all arguments.

---

### Example — First parameter separate, rest grouped

```javascript
function show(first, ...others) {
    console.log(first);   // 10
    console.log(others);  // [20, 30, 40]
}

show(10, 20, 30, 40);
```

---

# **3.2 Rest in Array Destructuring**

### Example:

```javascript
var arr = [10, 20, 30, 40];

var [a, ...rest] = arr;

console.log(a);    // 10
console.log(rest); // [20, 30, 40]
```

---

# **3.3 Rest in Object Destructuring**

```javascript
var student = {
    name: "Krishna",
    age: 18,
    branch: "CSE",
    city: "Ahmedabad"
};

var { name, ...details } = student;

console.log(name);     // Krishna
console.log(details);  // { age: 18, branch: "CSE", city: "Ahmedabad" }
```

---

# **4. Spread vs Rest — EASY TABLE**

| Feature | Spread                          | Rest                           |
| ------- | ------------------------------- | ------------------------------ |
| Purpose | Expand                          | Collect                        |
| Shape   | Breaks down                     | Gathers                        |
| Used In | Arrays, objects, function calls | Function params, destructuring |
| Example | `[...arr]`                      | `[a, ...rest]`                 |

---

# **5. Visual Understanding (Simple Diagram)**

### Spread (break apart)

```
arr = [1, 2, 3]

[ ...arr ] → 1, 2, 3
```

---

### Rest (group together)

```
[ a, ...rest ] = [1, 2, 3, 4]

rest → [2, 3, 4]
```

# **7. Common Mistakes**

---

### ❌ Mistake 1 — Using rest in wrong place

```javascript
...arr, 5  // ERROR
```

Rest must be the **last element**.

---

### ❌ Mistake 2 — Using spread with non-iterables

```javascript
...[10, 20]   // OK
...10         // ERROR (10 is not iterable)
```

---

### ❌ Mistake 3 — Wrong order in merging objects

```javascript
var a = { x: 1 };
var b = { x: 2 };

var result = { ...b, ...a }; // a overwrites b
```

Order matters!

---

### ❌ Mistake 4 — Confusing rest and spread

```javascript
var [...x, y] = arr; // ❌ invalid
```

Rest must come **last**.

# **8. Summary**

| Concept | Spread                          | Rest                           |
| ------- | ------------------------------- | ------------------------------ |
| Symbol  | ...                             | ...                            |
| Meaning | Expand values                   | Collect values                 |
| Used In | Arrays, Objects, Function calls | Function params, Destructuring |
| Memory  | Creates new copy (shallow)      | Groups data                    |
