# Array Methods in JavaScript

Arrays become powerful only when you learn the built-in methods that help you insert, remove, search, copy, and modify elements effectively.

This article covers all the array methods students must learn with clear examples.

---

# 1. push()

Adds one or more elements **to the end** of an array.

### Syntax:

```javascript
array.push(value);
```

### Example:

```javascript
var students = ["Jenil", "Mahir"];
students.push("Yashvi");

console.log(students); // ["Jenil", "Mahir", "Yashvi"]
```

---

# 2. pop()

Removes the **last element** and returns it.

### Example:

```javascript
var nums = [10, 20, 30];
var removed = nums.pop();

console.log(nums);     // [10, 20]
console.log(removed);  // 30
```

---

# 3. unshift()

Adds one or more elements **at the beginning**.

### Example:

```javascript
var list = [2, 3, 4];
list.unshift(1);

console.log(list); // [1, 2, 3, 4]
```

---

# 4. shift()

Removes the **first element** and returns it.

### Example:

```javascript
var arr = [100, 200, 300];
var first = arr.shift();

console.log(arr);   // [200, 300]
console.log(first); // 100
```

---

# 5. indexOf()

Returns the **first index** of an element, or -1 if not found.

### Example:

```javascript
var names = ["Jenil", "Arjun", "Krishna", "Arjun"];

console.log(names.indexOf("Arjun"));  // 1
console.log(names.indexOf("Mahir"));  // -1
```

---

# 6. lastIndexOf()

Returns the **last index** of an element.

```javascript
var names = ["A", "B", "C", "B", "D"];

console.log(names.lastIndexOf("B")); // 3
```

---

# 7. includes()

Checks if an array contains a specific value.
Returns **true** or **false**.

### Example:

```javascript
var ids = [101, 102, 103];

console.log(ids.includes(102)); // true
console.log(ids.includes(200)); // false
```

---

# 8. reverse()

Reverses the array **in-place** (changes original).

```javascript
var nums = [1, 2, 3];
nums.reverse();

console.log(nums); // [3, 2, 1]
```

---

# 9. sort() – Basic Version

Sorts elements **alphabetically** (as strings).

### Example:

```javascript
var students = ["Arjun", "Jenil", "Mahir", "Krishna"];
students.sort();

console.log(students);
```

Output:

```
["Arjun", "Jenil", "Krishna", "Mahir"]
```

### Important note :

Numbers may sort incorrectly because sort() converts them to strings.

```javascript
var nums = [10, 2, 5];
nums.sort(); 

console.log(nums); // [10, 2, 5] → Wrong order
```

---

# 10. join()

Converts an array into a **single string**, using a separator.

### Example:

```javascript
var subjects = ["HTML", "CSS", "JavaScript"];
var s = subjects.join(" - ");

console.log(s); // HTML - CSS - JavaScript
```

---

# 11. concat()

Combines two or more arrays into a new array.

### Example:

```javascript
var arr1 = [1, 2];
var arr2 = [3, 4];

var result = arr1.concat(arr2);

console.log(result); // [1, 2, 3, 4]
```

Original arrays remain unchanged.

---

# 12. slice()

Returns a **portion** of the array (does NOT modify original).

### Syntax:

```javascript
array.slice(start, end);
```

### Example:

```javascript
var nums = [10, 20, 30, 40, 50];

var part = nums.slice(1, 4);

console.log(part); // [20, 30, 40]
console.log(nums); // Original stays same
```

---

# 13. splice()

Modifies the array by removing or inserting elements.

### Syntax:

```javascript
array.splice(startIndex, deleteCount, item1, item2, ...)
```

### Example 1: Remove elements

```javascript
var arr = [10, 20, 30, 40];
arr.splice(1, 2);

console.log(arr); // [10, 40]
```

### Example 2: Insert elements

```javascript
var arr = [1, 4, 5];
arr.splice(1, 0, 2, 3);

console.log(arr); // [1, 2, 3, 4, 5]
```

### Example 3: Replace elements

```javascript
var arr = [100, 200, 300];
arr.splice(1, 1, 250);

console.log(arr); // [100, 250, 300]
```

---

# 14. toString()

Converts array to a comma-separated string.

```javascript
var arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

---

# 15. fill() – Basic Unit-1 Introduction

Fills all elements with a fixed value.

```javascript
var arr = [1, 2, 3, 4];
arr.fill(0);

console.log(arr); // [0, 0, 0, 0]
```

---

# 16. Real Example

### Store marks of 5 students and use array methods.

```javascript
var marks = [85, 90, 75, 92, 88];

// Add a new student's marks
marks.push(80);

// Remove lowest marks (last student)
marks.pop();

// Sort (string-based in Unit 1)
marks.sort();

console.log(marks); 
console.log("Top 3:", marks.slice(0, 3));
```

# 17. Common Mistakes

1. Forgetting that sort() sorts alphabetically
2. Confusing slice with splice
3. Expecting push/pop to modify the beginning (they don’t)
4. Thinking concat modifies original array (it doesn’t)
5. Using indexOf() on nested arrays
