
# FUNCTIONS IN JAVASCRIPT

# 1. What Is a Function?

A function is a reusable block of code designed to perform a specific task.

Instead of writing the same logic repeatedly, you write it once inside a function and call it whenever required.

Functions help in:

* Reusability
* Cleaner code
* Reduced repetition
* Better structure
* Easier debugging

---

# 2. Function Declaration

A **function declaration** defines a named function.

### Syntax:

```javascript
function functionName(parameter1, parameter2, ...) {
    // code to run
    return result;
}
```

### Example: Greeting a CodingGita Student

```javascript
function greet(name) {
    return "Welcome to CodingGita, " + name;
}

console.log(greet("Jenil"));
console.log(greet("Priyesha"));
```

### Line-by-line explanation:

1. `function greet(name)` → declares a function named `greet` with one parameter.
2. `return "Welcome to CodingGita, " + name;` → produces an output.
3. `greet("Jenil")` → calls the function.

### Key characteristics:

* Function declarations are **hoisted** (can be called before definition).

Example:

```javascript
console.log(add(2, 3)); // This works

function add(a, b) {
    return a + b;
}
```

---

# 3. Function Expression

A **function expression** stores a function inside a variable.

### Syntax:

```javascript
const variableName = function(parameter1, parameter2) {
    return result;
};
```

### Example:

```javascript
const percentage = function(marks, total) {
    return (marks / total) * 100;
};

console.log(percentage(85, 100));
```

### Differences from function declaration:

* **Not hoisted**
* Must be defined before calling
* Commonly used when passing functions as values

---

# 4. Parameters vs Arguments

### Parameters

Variables defined inside the function definition.

Example:

```javascript
function add(a, b) {
```

`a` and `b` are **parameters**.

### Arguments

Actual values passed while calling the function.

```javascript
add(10, 20);
```

`10` and `20` are **arguments**.

---

# 5. Return Statement

`return` sends a value back from the function.

Example:

```javascript
function area(length, width) {
    return length * width;
}

console.log(area(10, 5));
```

If a function does NOT return something, it returns `undefined`.

# 6. Why Use Functions? (Real-world Example)

### Without a function:

```javascript
let m1 = 85, m2 = 92, m3 = 78;
let avg1 = (m1 + m2 + m3) / 3;

let x1 = 70, x2 = 60, x3 = 80;
let avg2 = (x1 + x2 + x3) / 3;
```

Lots of repetitive code.

### With a function:

```javascript
function average(a, b, c) {
    return (a + b + c) / 3;
}

console.log(average(85, 92, 78)); // For Jenil
console.log(average(70, 60, 80)); // For Mahir
```

Cleaner, shorter, reusable.

# 7. Function Scope (Basic Introduction for Unit-1)

Inside a function, any variable you declare becomes **local** to that function.

### Example:

```javascript
function showMarks() {
    let marks = 90;
    console.log(marks);
}

showMarks();
console.log(marks); // Error: marks is not defined
```

* `marks` exists **only inside the function**.

### Global Scope:

Variables declared outside any function are accessible everywhere.

```javascript
let batch = "CSE";

function printBatch() {
    console.log(batch);
}

printBatch(); // CSE
```

# 8. Combining Function Concepts

### Problem:

Write a function that calculates total marks and average for a student.

### Solution:

```javascript
function analyzeStudent(name, m1, m2, m3) {
    const total = m1 + m2 + m3;
    const avg = total / 3;

    return name + " scored total " + total + " with average " + avg;
}

console.log(analyzeStudent("Yashvi", 85, 90, 92));
console.log(analyzeStudent("Arjun", 70, 75, 80));
```

---

# 9. Nested Functions (Functions inside functions)

A function can contain another function.

### Example:

```javascript
function studentReport(name, s1, s2, s3) {
    
    function total(a, b, c) {
        return a + b + c;
    }

    const t = total(s1, s2, s3);
    return name + " total marks: " + t;
}

console.log(studentReport("Krishna", 75, 88, 92));
```
