# Arrays in JavaScript

- An **array** is a special JavaScript data structure used to store a list of values.

- Arrays are used everywhere: storing student names, marks, items in a cart, search results, etc.

# 1. What Is an Array?

An array is an **ordered collection** of values.
Each value has a unique **index** (position).

Index always starts at **0**.

### Example:

```javascript
var numbers = [10, 20, 30, 40];
```

Here:

* 10 is at index 0
* 20 is at index 1
* 30 is at index 2
* 40 is at index 3

---

# 2. Creating Arrays

### 1. Using square brackets (recommended)

```javascript
var students = ["Jenil", "Mahir", "Priyesha", "Yashvi"];
```

### 2. Using the Array constructor (less common)

```javascript
var marks = new Array(85, 90, 78);
```

---

# 3. Accessing Values in an Array

Use **index number**.

```javascript
var students = ["Jenil", "Mahir", "Priyesha"];

console.log(students[0]); // Jenil
console.log(students[2]); // Priyesha
```

If you access an index that doesn’t exist:

```javascript
console.log(students[10]); // undefined
```

---

# 4. Updating Values

Simply assign to a specific index.

```javascript
var marks = [80, 75, 90];
marks[1] = 95;

console.log(marks); // [80, 95, 90]
```

---

# 5. Array Length

Every array has a built-in property called **length**.

```javascript
var arr = [1, 2, 3, 4];
console.log(arr.length); // 4
```

Length helps in loops.

---

# 6. Looping Through Arrays

### Method 1: for loop

```javascript
var students = ["Jenil", "Mahir", "Krishna", "Yashvi"];

for (var i = 0; i < students.length; i++) {
    console.log(students[i]);
}
```

### Method 2: while loop

```javascript
var i = 0;
while (i < students.length) {
    console.log(students[i]);
    i++;
}
```

---

# 7. Adding Elements to Arrays

### push() → Add at the end

```javascript
var nums = [1, 2, 3];
nums.push(4);

console.log(nums); // [1, 2, 3, 4]
```

### unshift() → Add at the beginning

```javascript
nums.unshift(0);
console.log(nums); // [0, 1, 2, 3, 4]
```

---

# 8. Removing Elements

### pop() → Remove last

```javascript
nums.pop();
console.log(nums);
```

### shift() → Remove first

```javascript
nums.shift();
console.log(nums);
```

---

# 9. Searching in Arrays (Unit-1 Basics)

### indexOf()

Returns index of a value, or -1 if not found.

```javascript
var names = ["Jenil", "Mahir", "Arjun"];

console.log(names.indexOf("Mahir")); // 1
console.log(names.indexOf("Krishna")); // -1
```

---

# 10. Array with Different Data Types

JavaScript arrays can store mixed types.

```javascript
var data = [100, "Hello", true, null];
```

---

# 11. Array of Arrays (Matrix)

Used in tables, grids, games.

```javascript
var matrix = [
    [1, 2],
    [3, 4],
    [5, 6]
];

console.log(matrix[2][1]); // 6
```

# 12. Real-World Example

### Store marks of 3 students:

```javascript
var marks = [
    [85, 90, 92], // Jenil
    [70, 75, 80], // Arjun
    [88, 82, 89]  // Krishna
];

for (var i = 0; i < marks.length; i++) {
    var studentMarks = marks[i];
    var total = studentMarks[0] + studentMarks[1] + studentMarks[2];
    var avg = total / 3;

    console.log("Student " + (i+1) + " average: " + avg);
}
```

# 13. Common Mistakes 

1. Forgetting index starts at 0
2. Using wrong index:

```javascript
arr[arr.length] = 10; // Wrong
```

3. Confusing arrays with objects
4. Expecting an error when accessing invalid index (returns undefined)
5. Creating sparse arrays accidentally:

```javascript
var a = [];
a[5] = 10; // Creates empty slots
```
