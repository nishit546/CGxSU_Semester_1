# **Destructuring Assignment**

Destructuring Assignment is an ES6 feature that allows you to **extract values** from arrays and objects and assign them to variables **in a clean, short, and readable way**.

Before ES6, extracting values was often repetitive and messy.
With destructuring, your code becomes cleaner, more readable, and more professional.

This is one of the MOST used ES6 features in modern JavaScript, React, Node.js, and real industry code.

---

# **1. What is Destructuring? (Simple Definition)**

Destructuring means:

> **Taking apart** arrays or objects and storing their values in separate variables easily.

Example:

### Old way (before ES6)

```javascript
var student = ["Arjun", 18, "CSE"];

var name = student[0];
var age = student[1];
var branch = student[2];
```

### ES6 Destructuring

```javascript
var [name, age, branch] = ["Arjun", 18, "CSE"];
```

Clean, short, readable.

---

# **2. Array Destructuring (Basics)**

### Example:

```javascript
var arr = ["Mahir", 19, "CSE"];

var [name, age, branch] = arr;

console.log(name);   // Mahir
console.log(age);    // 19
console.log(branch); // CSE
```

---

# **3. Skipping Elements in Array Destructuring**

You can skip values by leaving empty commas.

```javascript
var marks = [90, 85, 78, 92];

var [first, , third] = marks;

console.log(first); // 90
console.log(third); // 78
```

---

# **4. Default Values in Array Destructuring**

If the value is missing, use default:

```javascript
var arr = ["Krishna"];

var [name, age = 18] = arr;

console.log(name); // Krishna
console.log(age);  // 18 (default)
```

---

# **5. Swapping Variables with Destructuring (Most Popular Use Case)**

The classic swap problem becomes one line:

```javascript
var a = 10;
var b = 20;

[a, b] = [b, a];

console.log(a, b); // 20 10
```

No temporary variable needed.

---

# **6. Rest Operator in Array Destructuring**

The rest operator `...` gathers remaining elements.

```javascript
var students = ["Jenil", "Arjun", "Krishna", "Yashvi"];

var [first, second, ...others] = students;

console.log(first);   // Jenil
console.log(second);  // Arjun
console.log(others);  // ["Krishna", "Yashvi"]
```

---

# **7. Nested Array Destructuring**

Works even for multi-dimensional (nested) arrays.

```javascript
var nums = [1, [2, 3], 4];

var [a, [b, c], d] = nums;

console.log(a, b, c, d); // 1 2 3 4
```

---

# **8. Real Example — Extracting Marks**

```javascript
var student = ["Priyesha", [88, 92, 79]];

var [name, [m1, m2, m3]] = student;

console.log(name); // Priyesha
console.log(m1, m2, m3);
```

---

# **9. Object Destructuring (Basics)**

Object destructuring extracts properties from objects into variables.

### Example:

```javascript
var student = {
    name: "Arjun",
    age: 19,
    branch: "CSE"
};

var { name, age, branch } = student;

console.log(name, age, branch);
```

---

# **10. Order Doesn't Matter in Object Destructuring**

Object destructuring works by **matching keys**, not order.

```javascript
var { age, name } = student;

console.log(name, age); // Arjun 19
```

---

# **11. Renaming Variables in Object Destructuring**

```javascript
var student = {
    name: "Krishna",
    age: 18
};

var { name: studentName, age: studentAge } = student;

console.log(studentName); // Krishna
console.log(studentAge);  // 18
```

---

# **12. Default Values in Object Destructuring**

```javascript
var student = { name: "Yashvi" };

var { name, branch = "CSE" } = student;

console.log(branch); // CSE (default)
```

---

# **13. Rest Operator in Object Destructuring**

```javascript
var student = {
    name: "Mahir",
    age: 19,
    branch: "CSE",
    city: "Ahmedabad"
};

var { name, age, ...otherInfo } = student;

console.log(otherInfo);
// { branch: "CSE", city: "Ahmedabad" }
```

---

# **14. Nested Object Destructuring**

```javascript
var student = {
    name: "Jenil",
    marks: {
        math: 90,
        physics: 88
    }
};

var { name, marks: { math, physics } } = student;

console.log(math, physics); // 90 88
```

---

# **15. Destructuring in Function Parameters (Very Powerful)**

### Before (old way):

```javascript
function show(student) {
    console.log(student.name);
    console.log(student.age);
}
```

### With destructuring:

```javascript
function show({ name, age }) {
    console.log(name);
    console.log(age);
}

show({ name: "Krishna", age: 18 });
```

Makes functions cleaner.

---

# **16. Function Parameter Defaulting with Destructuring**

```javascript
function enroll({ name = "Student", branch = "CSE" } = {}) {
    console.log(name, branch);
}

enroll(); // Student CSE
```

---

# **17. Mixed Destructuring (Object + Array)**

```javascript
var data = {
    name: "Arjun",
    marks: [90, 88, 92]
};

var { name, marks: [m1, m2, m3] } = data;

console.log(name, m1, m2, m3);
```

# **19. Common Beginner Mistakes**

### ❌ Mistake 1 — Using wrong variable names

```javascript
var { x } = { y: 10 }; // x is undefined
```

---

### ❌ Mistake 2 — Forgetting brackets when destructuring objects

```javascript
var name;
{name} = student; // ERROR
```

JavaScript thinks `{}` is a block.

Fix using parentheses:

```javascript
({ name } = student);
```

---

### ❌ Mistake 3 — Destructuring undefined values

```javascript
var { age } = null; // ERROR
```

---

### ❌ Mistake 4 — Typos

Destructuring fails silently:

```javascript
var { branc } = student; // undefined
```

# **22. MCQs**

### **Q1. Which syntax is correct?**

1. `var [a, b] = arr;`
2. `var {a b} = obj;`
3. `var [a: b] = arr;`
4. `var a, b = arr;`

**Answer: 1**

---

### **Q2. What happens if a property is missing?**

```javascript
var { x } = { };
```

1. Error
2. x = undefined
3. x = null
4. x = 0

**Answer: 2**

---

### **Q3. Which operator collects remaining values?**

1. spread
2. rest
3. join
4. map

**Answer: 2**

---

### **Q4. What is the result?**

```javascript
var [a, , c] = [1, 2, 3];
```

**Answer:** `a = 1`, `c = 3`
