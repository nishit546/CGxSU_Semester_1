
# **1. Subgraphs** üìâ

**Definition:**
A **subgraph** is a smaller portion of a larger graph, formed by selecting some vertices (nodes) and edges (connections) from the original graph. Formally, for a graph (G = (V, E)), a subgraph (G' = (V', E')) satisfies:

* (V' \subseteq V) ‚úÖ
* (E' \subseteq E) ‚úÖ

Subgraphs can be **induced** (all edges among chosen vertices included) or **non-induced** (some edges included). They are used to analyze parts of a network or simplify complex graphs.

---

### **Examples of Subgraphs:**

**Original Graph (G):**

```
      A
     / \
    B---C
   /     \
  D       E
```

---

**Example 1:**

* **Vertices:** {A, B, C}
* **Edges:** {A-B, B-C}
* **Explanation:** Subgraph includes 3 vertices and 2 edges.
* **Visual:**

```
      A
       \
        B---C
```

---

**Example 2:**

* **Vertices:** {B, D}
* **Edges:** {B-D}
* **Explanation:** A simple edge connecting two nodes.
* **Visual:**

```
   B
   |
   D
```

---

**Example 3 (Induced Subgraph) üü¢:**

* **Vertices:** {B, C, E}
* **Edges automatically included:** {B-C, C-E}
* **Visual:**

```
   B---C
        \
         E
```

---

**Example 4:**

* **Vertices:** {A, E}
* **Edges:** none
* **Explanation:** Subgraph can have vertices without edges.
* **Visual:**

```
A     E
```

---

**Example 5 (Non-Induced Subgraph) ‚ö†Ô∏è:**

* **Vertices:** {C, D, E}
* **Edges:** {C-E}
* **Explanation:** Only some edges included.
* **Visual:**

```
   C---E
   D
```

---

**Example 6:**

* **Vertices:** {A, B, D}
* **Edges:** {A-B, B-D}
* **Visual:**

```
      A
       \
        B
        |
        D
```

---

‚úÖ **Summary:**

* Subgraphs can be **single nodes**, **connected portions**, or **disconnected vertices**.
* **Induced subgraphs** automatically include all edges among chosen vertices.
* Useful in **network analysis** and **pattern recognition**.

# **2. Connected Components** üîó

**Definition:**
A **connected component** of an undirected graph is a set of vertices where:

1. Every vertex is reachable from every other vertex in the set üåê
2. No edges connect to vertices outside the set üö´

**Key Point:**

* A connected graph has **1 component**; others may have multiple.

---

### **Examples of Connected Components:**

**Graph (G):**

```
      A
     / \
    B---C

      D---E

      F
```

---

**Example 1:**

* **Component 1:** {A, B, C}
* **Explanation:** All vertices connected internally.
* **Visual:**

```
   A
  / \
 B---C
```

---

**Example 2:**

* **Component 2:** {D, E}
* **Explanation:** Isolated cluster.
* **Visual:**

```
D---E
```

---

**Example 3:**

* **Component 3:** {F}
* **Explanation:** Isolated vertex.
* **Visual:**

```
F
```

---

**Example 4:**

* **Graph:**

```
  P---Q   R---S---T
          U
```

* **Connected Components:**

  1. {P, Q} ‚úÖ
  2. {R, S, T, U} ‚úÖ

---

**Example 5:**

* **Graph:**

```
  1---2---3   4---5   6
```

* **Connected Components:**

  1. {1, 2, 3}
  2. {4, 5}
  3. {6}

---

**Example 6:**

* **Graph:**

```
  X---Y   Z
```

* **Connected Components:**

  1. {X, Y}
  2. {Z}
