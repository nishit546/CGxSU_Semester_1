## üîπ 1Ô∏è‚É£ Determinant of a Matrix ‚Äî Concept & Explanation

A **determinant** is a scalar value computed from a square matrix. It helps determine whether a matrix has an inverse.

For a square matrix **A**, the determinant is denoted as **|A|** or **det(A)**.

### üí° Key Points:

1. Determinants exist **only for square matrices** (n √ó n).
2. If **|A| = 0**, the matrix is **singular** (no inverse exists).
3. If **|A| ‚â† 0**, the matrix is **non-singular** (inverse exists).

---

### üìä Visual Representation (2√ó2):

A = [ a b ]
[ c d ]

det(A) = ad - bc

---

### üß† Examples (2√ó2 Matrices)

**Example 1:**

A = [ 2 3 ]
[ 1 4 ]
det(A) = 8 - 3 = 5

**Example 2:**

B = [ 5 7 ]
[ 2 6 ]
det(B) = 30 - 14 = 16

**Example 3:**

C = [ 1 2 ]
[ 3 4 ]
det(C) = 4 - 6 = -2

**Example 4 (Singular Matrix):**

D = [ 3 5 ]
[ 6 10 ]
det(D) = 30 - 30 = 0

**Example 5:**

E = [ 7 2 ]
[ 4 1 ]
det(E) = 7 - 8 = -1

**Example 6:**

F = [ 8 3 ]
[ 2 9 ]
det(F) = 72 - 6 = 66

---

### üß† Example (3√ó3 Matrix)

A = [ 1 2 3 ]
[ 0 1 4 ]
[ 5 6 0 ]

Determinant formula for 3√ó3:

det(A) = a(ei - fh) - b(di - fg) + c(dh - eg)

Substitute values:

det(A) = 1*(10 - 46) - 2*(00 - 45) + 3*(06 - 15)
det(A) = -24 + 40 - 15
det(A) = 1

‚úÖ Determinant = 1

## üîπ 2Ô∏è‚É£ Inverse of a Matrix ‚Äî Concept & Explanation

The **inverse of a matrix** is a matrix that, when multiplied with the original matrix, gives the **identity matrix (I)**:

A √ó A‚Åª¬π = I

- Only **square matrices** with **|A| ‚â† 0** have inverses.
- Inverse is useful to solve linear equations:  
`AX = B ‚Üí X = A‚Åª¬πB`.

---

### üí° Key Formula (2√ó2 matrix)

A = [ a b ]
[ c d ], det(A) ‚â† 0

A‚Åª¬π = (1/det(A)) * [ d -b ]
[ -c a ]

---

### üß† Examples (2√ó2 Matrices)

**Example 1:**

A = [ 2 3 ]
[ 1 4 ]
det(A) = 5

A‚Åª¬π = (1/5) * [ 4 -3 ]
[ -1 2 ]
= [ 0.8 -0.6 ]
[ -0.2 0.4 ]

**Example 2:**

B = [ 1 2 ]
[ 3 4 ]
det(B) = -2

B‚Åª¬π = (1/-2) * [ 4 -2 ]
[ -3 1 ]
= [ -2 1 ]
[ 1.5 -0.5 ]

---

### üß† Example (3√ó3 Matrix Inverse)

A = [ 1 2 3 ]
[ 0 1 4 ]
[ 5 6 0 ]

**Step 1: Determinant**

det(A) = 1

**Step 2: Cofactor matrix**

C = [ -24 20 -5 ]
[ 18 -15 4 ]
[ 5 -4 1 ]

**Step 3: Adjugate (transpose of cofactor)**

adj(A) = [ -24 18 5 ]
[ 20 -15 -4 ]
[ -5 4 1 ]

**Step 4: Inverse**

A‚Åª¬π = (1/det(A)) * adj(A)
= 1 * adj(A)
= [ -24 18 5 ]
[ 20 -15 -4 ]
[ -5 4 1 ]

‚úÖ This is the inverse of the 3√ó3 matrix.

---

### ‚ú® Key Notes:

- A matrix has an inverse **only if det(A) ‚â† 0**.
- The **identity matrix** acts as ‚Äú1‚Äù in multiplication.
- The inverse is used to solve linear systems `AX = B ‚Üí X = A‚Åª¬πB`.
- For **3√ó3 matrices**, the inverse uses the **adjugate/adjoint method**.

---

| 2√ó2 matrix | ad - bc                    | (1/det) * [ d -b ; -c a ] |

## üîπ 3Ô∏è‚É£ Gauss-Jordan Method to Find Inverse

Gauss-Jordan Method is a systematic way to find the inverse of a matrix using **row operations** instead of the adjoint method.

---

### **Steps (Rules)**

1. **Augment** the square matrix with the identity matrix: `[A | I]`.
2. Perform **row operations** until the **left side becomes identity**.
3. **The right side becomes the inverse** \(A‚Åª¬π\).

**Row operations allowed:**

1. Swap two rows ‚Üí `R_i ‚Üî R_j`
2. Multiply a row by a non-zero constant ‚Üí `k R_i`
3. Add/subtract multiple of one row to another ‚Üí `R_j ‚Üí R_j + k R_i`

> Always do the **same operation on both sides** of the augmented matrix.

---

### üß† Example: 2√ó2 Matrix (Gauss-Jordan)

A = [ 2 1 ]
[ 3 2 ]

**Step 1: Augment with identity**

[A | I] = [ 2 1 | 1 0 ]
[ 3 2 | 0 1 ]

**Step 2: Make pivot 1 (R1 √∑ 2)**

[ 1 0.5 | 0.5 0 ]
[ 3 2 | 0 1 ]

**Step 3: Make zero below pivot (R2 - 3√óR1)**

[ 1 0.5 | 0.5 0 ]
[ 0 0.5 | -1.5 1 ]

**Step 4: Make second pivot 1 (R2 √∑ 0.5)**

[ 1 0.5 | 0.5 0 ]
[ 0 1 | -3 2 ]

**Step 5: Make zero above second pivot (R1 - 0.5√óR2)**

[ 1 0 | 2 -1 ]
[ 0 1 | -3 2 ]

‚úÖ **Inverse:**

A‚Åª¬π = [ 2 -1 ]
[ -3 2 ]

---

### üß† Example: 3√ó3 Matrix (Gauss-Jordan)

A = [ 2 1 1 ]
[ 1 3 2 ]
[ 1 0 0 ]

**Step 1: Augment with identity**

[A | I] = [ 2 1 1 | 1 0 0 ]
[ 1 3 2 | 0 1 0 ]
[ 1 0 0 | 0 0 1 ]


**Step 2: Reduce to identity (pivot by pivot, using row operations)**

- Step-by-step row operations lead to:

[ 1 0 0 | 0 0 1 ]
[ 0 1 0 | -2 1 3 ]
[ 0 0 1 | 3 -1 -5 ]

‚úÖ **Inverse:**

A‚Åª¬π = [ 0 0 1 ]
[ -2 1 3 ]
[ 3 -1 -5 ]

---

### ‚ú® Key Notes

- A matrix has an inverse **only if det(A) ‚â† 0**.  
- The **identity matrix** acts as ‚Äú1‚Äù in multiplication.  
- The **Gauss-Jordan method** is systematic and works for both 2√ó2 and 3√ó3 matrices.  
- For 3√ó3 matrices, **row operations** are often easier than adjoint method for manual inverses.  
- Always verify: `A √ó A‚Åª¬π = I`.
